# 客户审查准备

## 工作流程

### 步骤1：客户背景

收集或查询：
- **客户姓名**和家庭成员
- **账户类型**：应税、IRA、罗斯、401(k)、信托等
- **总管理资产规模**跨账户
- **投资政策声明（IPS）**：目标配置、风险承受能力、约束条件
- **人生阶段**：积累、退休前、退休、遗产
- **上次会议日期**和任何未完成的行动项

### 步骤2：投资组合业绩

对每个账户和家庭总合：

| 指标 | 季度至今 | 年初至今 | 1年 | 3年 | 成立以来 |
|------|---------|---------|------|------|----------|
| 投资组合收益 | | | | | |
| 基准收益 | | | | | |
| 阿尔法 | | | | | |

**业绩归因：**
- 哪些资产类别/持仓驱动了收益？
- 前3大贡献者和前3大拖累
- 任何单一持仓的巨大影响？

### 步骤3：配置审查

当前与目标配置：

| 资产类别 | 目标 | 当前 | 偏离 | 行动 |
|---------|------|------|------|------|
| 美国大盘股 | | | | |
| 美国中小盘股 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 固定收益 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记任何超出IPS再平衡阈值的偏离（通常为3-5%）。

### 步骤4：讨论要点

生成会议议程：

1. **市场概览**（2-3分钟）：简要宏观背景和展望
2. **投资组合业绩**（5分钟）：我们表现如何？为什么？
3. **配置审查**（5分钟）：是否需要再平衡？
4. **规划更新**（5-10分钟）：
   - 生活变化？（工作、健康、家庭、住房、教育）
   - 收入需求变化？
   - 税务情况更新
   - 遗产规划更新
5. **行动项**（5分钟）：下次会议前我们要做什么？

### 步骤5：主动建议

基于审查，建议：
- 再平衡交易（如果偏离超过阈值）
- 税损收获机会
- 现金部署或提款规划
- 罗斯转换机会（如适用）
- 受益人更新或遗产规划需求
- 保险审查（人寿、伤残、长期护理）

### 步骤6：输出

- 单页客户审查摘要（Word或PDF）
- 带基准的业绩表
- 配置饼图（当前 vs. 目标）
- 建议的行动项
- 会议议程

## 重要注意事项

- 会议前了解你的客户 — 回顾上次会议的记录
- 以客户关心的内容为导向，而非你想讨论的内容
- 如果业绩不佳，直接解决 — 不要隐藏或粉饰
- 始终以明确的行动项和带日期的后续步骤结束
- 记录会议记录和IPS的任何变更
- 合规：确保所有材料符合公司政策和监管要求
