# 财务计划

## 工作流程

### 步骤1：客户资料

收集或确认：
- **人口统计信息**：年龄、配偶年龄、受抚养人、预期寿命假设
- **就业情况**：当前收入、预期加薪、目标退休年龄
- **账户**：所有投资账户的余额和资产配置
- **收入来源**：工资、奖金、租金收入、社会保障估算、养老金
- **支出**：当前年度支出、预期变化（抵押贷款还清、子女独立）
- **负债**：抵押贷款、学生贷款、其他债务
- **保险**：人寿、伤残、长期护理、健康保险
- **遗产**：遗嘱、信托、受益人指定、赠与策略

### 步骤2：现金流分析

构建年度现金流预测：

| 年份 | 年龄 | 总收入 | 税费 | 生活支出 | 储蓄 | 净现金流 |
|------|------|--------|------|----------|------|----------|
|      |      |        |      |          |      |          |

关键输入：
- 通胀率假设（通常为2.5-3%）
- 税率（边际税率和有效税率）
- 储蓄率及储蓄去向（税前、罗斯、应税）

### 步骤3：退休规划

**积累阶段：**
- 当前投资组合价值
- 年度供款（401k、IRA、应税账户）
- 按资产类别预期收益
- 蒙特卡洛模拟：不同支出水平下的成功概率

**分配阶段：**
- 退休后所需年度支出（当前美元 → 通胀调整后）
- 社会保障开始年龄和福利
- 养老金收入（如有）
- 投资组合提取率和顺序
-  Required Minimum Distributions (RMDs，最低必需分配)

**关键输出：**
- 退休时的预计投资组合价值
- 可持续提取率
- 资金不耗尽的概率（目标 >85%）
- "如果"情景：提前退休、市场下跌、更高支出

### 步骤4：目标特定分析

#### 教育资金
- 子女年龄和目标大学入学时间
- 当前529账户余额
- 目标资金水平（公立 vs. 私立，本科 vs. 研究生）
- 实现目标所需的月度储蓄
- 助学金考虑因素

#### 遗产规划
- 当前遗产价值和预计增长
- 遗产税风险（联邦和州）
- 已建立的信托结构
- 赠与策略（年度免税额、终身豁免使用）
- 慈善捐赠计划
- 受益人审查

#### 风险管理
- 人寿保险需求分析（收入替代、债务还清、教育资金）
- 伤残保险充足性
- 长期护理规划
- 伞式责任保险

|------|----------|------------------|------|

| 更高支出（+20%） | | | |
| 一方配偶活到95岁 | | | |
| 长期护理事件 | | | |

### 步骤6：建议

优先行动项：
1. 储蓄率调整
2. 资产配置调整
3. 税务优化（罗斯转换、税损收获、资产配置）
4. 保险缺口填补
5. 遗产文件更新
6. 受益人指定审查

### 步骤7：输出

- 财务计划文档（Word/PDF，15-25页）
- 现金流预测电子表格（Excel）
- 退休预测图表
- 目标资金分析
- 情景比较表
- 行动项清单

## 重要注意事项

- 财务计划是活文档 —— 每年或重大生活事件后审查和更新
- 收益假设要保守 —— 高估收益会产生错误信心
- 税务规划与投资收益同等重要 —— 为每项建议建模税务影响
- 社会保障 timing 是一个重要杠杆 —— 建模62岁、67岁和70岁的开始年龄
- 始终对计划进行压力测试 —— 只在基准情景下有效的计划不是好计划
- 合规：确保建议符合适合性/受托标准
