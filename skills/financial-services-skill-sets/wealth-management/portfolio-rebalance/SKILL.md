# 投资组合再平衡

## 工作流程

### 步骤1：当前状态

对每个账户，捕获：
- 账户类型（应税、IRA、罗斯、401k）
- 持有资产及其当前市场价值
- 成本基础（对应税账户）
- 每个持仓的未实现收益/损失

### 步骤2：偏离分析

将当前配置与投资政策声明目标进行比较：

| 资产类别 | 目标比例 | 当前比例 | 偏离 | 超配/低配金额 |
|---------|---------|---------|------|---------------|
| 美国大盘股 | | | | |
| 美国中小盘股 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 投资级债券 | | | | |
| 高收益/信贷 | | | | |
| 通胀保值债券 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记超出再平衡带宽的持仓（通常为±3-5%）。

### 步骤3：交易建议

生成交易以将配置恢复到目标水平：

**税务意识再平衡规则：**
- 优先在税收优惠账户（IRA、罗斯）中进行再平衡 — 无税务后果
- 在应税账户中，避免出售有大量短期收益的持仓
- 在再平衡的同时尽可能收获损失
- 注意所有账户的洗售规则（30天窗口）
- 考虑将新资金直接投入低配资产类别，而不是进行交易

**交易清单：**

| 账户 | 操作 | 证券 | 股数/金额 | 原因 | 税务影响 |
|------|------|------|-----------|------|----------|
|      | 买入/卖出 | | | 再平衡 / 税损收获 | 短期收益 / 长期收益 / 损失 |

### 步骤4：资产配置审查

优化哪些资产应持有在哪些账户类型中：
- **税收递延（IRA/401k）**：债券、REITs、高换手率基金（最高税务拖累）
- **罗斯**：预期增长最高的资产（免税增长）
- **应税**：税务高效的股票（指数基金、ETF、市政债券）、税损收获候选资产

### 步骤5：实施

- 按账户分类的总交易量
- 估计交易成本
- 估计税务影响（已实现收益/损失）
- 对配置偏离的净影响

### 步骤6：输出

- 偏离分析表
- 建议交易清单（Excel）
- 税务影响摘要
- 再平衡前后配置比较

## 重要注意事项

- 不要为了再平衡而再平衡 — 带宽内的小偏离是可以接受的
- 在应税账户中，税务成本可能超过再平衡收益 — 计算收支平衡点
- 在交易前考虑待处理的现金流（供款、提款、RMDs）
- 检查任何客户特定限制（ESG、集中持股、锁定期）
- 为每项交易记录理由，用于合规记录
- 洗售规则适用于所有账户 — 协调整个家庭的交易
