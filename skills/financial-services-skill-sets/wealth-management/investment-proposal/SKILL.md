# 投资方案

## 工作流程

### 步骤1：潜在客户背景

收集：
- **潜在客户姓名**和家庭详情
- **当前情况**：现有顾问？自主管理？什么促使了这次会议？
- **资产**：估计管理资产规模、账户类型、当前持有（如有分享）
- **目标**：退休、财富保值、增长、收入、教育、遗产
- **风险承受能力**：保守、适中、激进（或问卷评分）
- **约束条件**：ESG偏好、集中持股、流动性需求
- **费用敏感性**：他们现在支付什么费用？
- **竞争**：他们还在考虑谁？

### 步骤2：方案结构

**I. 关于我们公司**（1页）
- 公司概述、历史、管理资产规模
- 投资理念（用通俗易懂的语言）
- 团队简介（与该客户相关的）
- 客户服务模式（我们多久会面一次，他们联系谁）

**II. 了解您的需求**（1页）
- 重申他们的目标和关注点 — 展示您有倾听
- 发现过程中识别的关键规划考虑因素
- 对他们来说成功是什么样子

**III. 建议投资策略**（2-3页）
- 建议资产配置及理由
- 配置如何映射到他们的目标和风险承受能力
- 投资工具（ETF、共同基金、个股、另类投资）
- 税务意识策略（资产配置、税损收获）

建议配置：

| 资产类别 | 配置比例 | 投资工具 | 理由 |
|---------|---------|---------|------|
|         |         |         |      |

**IV. 预期结果**（1-2页）
- 预计增长情景（保守、适中、乐观）
- 实现目标的蒙特卡洛概率
- 收入预测（如果以退休或收入为重点）
- 风险指标（最大回撤、波动性）
- 与当前投资组合的比较（如有已知）

**V. 费用结构**（1页）
- 咨询费时间表（如适用的分层结构）
- 底层基金费用
- 总综合成本估算
- 费用与行业平均水平的比较
- 价值主张 — 他们从费用中获得什么

**VI. 开始流程**（1页）
- 账户开设流程
- 资产转移时间表
- 过渡计划（如果从其他顾问处转移）
- 前90天 — 预期内容
- 所需文件和后续步骤

### 步骤3：定制化

- 根据潜在客户调整语气（企业高管 vs. 小企业主 vs. 退休人员）
- 如果他们有集中持股，直接解决这个问题
- 如果他们将您与机器人顾问比较，强调规划和关系价值
- 如果他们对价格敏感，以总价值和结果为导向，而不仅仅是费用

### 步骤4：输出

- 带有公司品牌的PowerPoint演示文稿（12-15张幻灯片）
- PDF留存版本
- 后续邮件的单页摘要

## 重要注意事项

- 方案应感觉个性化，而非模板化 — 参考他们的具体情况
- 不要过度承诺业绩 — 设定现实期望并强调流程
- 始终包含免责声明（预测是假设的，过去业绩等）
- 过渡计划很重要 — 客户担心更换顾问的 disruption
- 在48小时内跟进方案并明确后续步骤
- 方案在向潜在客户展示前必须经过合规审查
