# 税损收获

## 工作流程

### 步骤1：识别候选资产

扫描应税账户中具有未实现损失的持仓：

| 证券 | 资产类别 | 成本基础 | 当前价值 | 未实现损失 | 持有期 | 损失百分比 |
|------|---------|---------|---------|-----------|--------|------------|
| | | | | | 短期/长期 | |

**优先顺序：**
1. 绝对损失最大（税务 benefit 最大）
2. 先处理短期损失（抵消按普通收入税率征税的短期收益）
3. 损失百分比最大的持仓（不太可能快速恢复）

### 步骤2：收益/损失预算

计算客户的税务情况：

| 类别 | 金额 |
|------|------|
| 年初至今已实现短期收益 | |
| 年初至今已实现长期收益 | |
| 年初至今已实现损失 | |
| 净收益/(损失)状况 | |
| 前几年结转的损失 | |
| **目标收获金额** | |

**税务节省估算：**
- 短期损失 × 边际普通收入税率
- 长期损失 × 资本利得税率
- 最多 $3,000 净损失可抵扣普通收入
- 超额部分结转

### 步骤3：替代证券

为每个收获候选资产建议替代证券，要求：
- 保持相似的市场敞口（相同资产类别、行业、地理区域）
- 不是"实质相同"（洗售规则）
- 具有相似的风险/收益特征

| 卖出 | 替换为 | 原因 | 跟踪误差风险 |
|------|--------|------|--------------|
| SPDR S&P 500 (SPY) | iShares Core S&P 500 (IVV) | 相同指数，不同基金家族 | 最小 |
| Vanguard Total Intl (VXUS) | iShares MSCI ACWI ex-US (ACWX) | 相似敞口，不同指数 | 低 |
| 个股 ABC | 行业 ETF (XLK) | 更广泛敞口，无洗售风险 | 中等 |

### 步骤4：洗售检查

执行前，验证无洗售：

- 检查家庭中的所有账户（应税、IRA、罗斯、配偶账户）
- 30天回溯：过去30天是否购买了实质相同的证券？
- 30天前瞻：30天内禁止回购相同证券
- 检查可能触发洗售的股息再投资计划（DRIPs）
- 记录每项交易的洗售窗口

| 卖出证券 | 洗售窗口开始 | 窗口结束 | DRIP 活跃？ | 风险 |
|---------|-------------|---------|------------|------|
| | | | | |

### 步骤5：执行计划

| 交易编号 | 账户 | 操作 | 证券 | 股数 | 估计 proceeds | 估计损失 | 替代证券 | 备注 |
|---------|------|------|------|------|--------------|----------|----------|------|
| | | 卖出 | | | | | | |
| | | 买入 | | | | | | |

**摘要：**
- 估计收获的总损失：$
- 估计税务节省：$（按 % 的边际税率）
- 投资组合净影响：最小（替代证券保持敞口）
- 洗售窗口管理：[日期]

### 步骤6：收获后跟踪

30+ 天后，可选择：
- 换回原证券（如果偏好）
- 维持替代证券（如果无理由换回）
- 更新成本基础记录
- 为税务报告记录文档

### 步骤7：输出

- 收获机会列表（Excel）
- 交易执行表
- 洗售跟踪日历
- 税务节省估算摘要
- 替代证券理由

## 重要注意事项

- 洗售规则严格 — 违规会禁止损失并调整成本基础
- 实质相同指相同证券，而非相同资产类别 — 跟踪不同指数的 ETF 通常没问题
- 始终协调整个家庭的所有账户，包括退休账户
- 考虑长期成本基础递减 — 收获会重置成本基础，意味着以后会有更多收益
- 年末是收获的黄金季节，但全年都存在机会
- 12月共同基金资本利得分配可能增加收获的紧迫性
- 为税务报告和合规记录所有内容
- 并非所有损失都值得收获 — 交易成本和跟踪误差有实际成本
