# 客户报告

## 工作流程

### 步骤1：报告参数

- **客户姓名**和家庭
- **报告期间**：季度、年初至今、年度、自定义范围
- **账户**：所有账户或特定账户
- **基准**：标普500、60/40混合、匹配IPS的自定义基准
- **公司品牌**：标志、颜色、免责声明

### 步骤5：市场评论

根据客户的专业水平定制的简要市场摘要：
- 本季度市场发生了什么（2-3句话）
- 这如何影响了投资组合
- 展望和定位理由（2-3句话）
- 对零售客户避免行话；对成熟投资者可以更技术化

### 步骤6：活动摘要

- 期间执行的交易
- 供款和提款
- 收到的股息和利息
- 收取的费用
- 再平衡活动

### 步骤7：规划说明

- 财务目标的进展（退休、教育等）
- 任何计划变更或建议
- 即将到来的行动项
- 下次审查日期

### 步骤8：输出

- 带公司品牌的PDF报告（8-12页）
- 用于定制的Word文档
- Excel数据附录（可选）

**报告结构：**
1. 封面（客户姓名、期间、公司标志）
2. 执行摘要（1页）
3. 业绩摘要（1-2页）
4. 带图表的配置概览（1页）
5. 持仓明细（1-2页）
6. 市场评论（1页）
7. 活动摘要（1页）
8. 规划说明（1页）
9. 披露和免责声明（1页）

## 重要注意事项

- 业绩计算必须扣除费用，除非客户/合规要求毛收益
- 始终包含适当的免责声明和披露（过去业绩、风险因素）
- 报告应在客户间保持一致 — 使用标准模板
- 根据客户匹配详细程度 — 有些客户想要每个持仓，有些想要单页摘要
- 基准选择很重要 — 使用IPS中的基准，而非看起来最好的基准
- 在新模板首次分发前，审查并获得合规批准
