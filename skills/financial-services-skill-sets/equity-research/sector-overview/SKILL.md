# 行业概览

## 工作流程

### 步骤1：界定范围

- **行业/子行业**：什么行业，定义有多窄？
- **目的**：客户报告、内部研究、推介材料、想法生成
- **深度**：高层概览（5-10页）还是深度研究（20-30页）
- **角度**：中性全景还是主题论点（如"AI基础设施建设"）
- **范围**：仅上市公司，还是包含非上市公司？

### 步骤2：市场概览

**市场规模与增长**
- 总可寻址市场（TAM）及来源
- 历史增长率（5年CAGR）
- 预测增长率及关键假设
- 市场细分（按产品、地区、终端市场、客户类型）

**行业结构**
- 分散vs集中——前5大市场份额
- 价值链图谱——价值在哪里积累？
- 商业模式类型（订阅、交易、许可、服务）
- 进入壁垒（资本、监管、技术、网络效应）

**关键趋势与驱动因素**
- 长期顺风因素（3-5个主要趋势）
- 逆风因素和风险
- 技术颠覆方向
- 监管发展
- 并购活动和整合趋势

### 步骤3：竞争格局

**公司概况**（前5-10大参与者）：

| 公司 | 收入 | 增长率 | EBITDA利润率 | 市场份额 | 关键差异化因素 |
|------|------|--------|-------------|---------|---------------|
| | | | | | |

每家公司的简要概况：
- 业务描述（2-3句话）
- 战略定位和护城河
- 近期发展（业绩、并购、产品发布）
- 估值快照（市盈率、EV/EBITDA、EV/收入）

**竞争动态**
- 公司如何竞争？（价格、产品、服务、渠道）
- 谁在获取/失去份额，为什么？
- 新进入者或相邻参与者的颠覆风险

### 步骤4：估值背景

- 行业交易倍数（当前和历史区间）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 该行业与大盘相比如何？

### 步骤5：投资启示

- 最佳风险/回报机会在哪里？
- 可以通过该行业表达哪些主题押注？
- 行业内的关键争论（多头vs空头论点）
- 可能改变行业叙事的催化剂

### 步骤6：输出

- Word文档或PowerPoint，包含：
  - 市场概览和规模测算
  - 竞争格局图谱
  - 公司对比表
  - 估值总结
  - 关键图表：市场增长、份额趋势、估值历史
- Excel附录，包含详细公司数据

## 重要说明

- 为所有市场规模数据注明来源——引用研究公司或方法论
- 区分TAM炒作和现实的可寻址市场
- 行业概览时效性强——注明日期并标记可能过时的数据
- 图表至关重要——市场规模瀑布图、竞争定位矩阵、估值散点图
- 如为客户准备，根据其具体情况调整"那又怎样"（并购目标识别、竞争定位、市场进入）
