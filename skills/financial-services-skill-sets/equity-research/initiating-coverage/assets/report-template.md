# 股票研究首次覆盖报告模板

本模板提供创建综合股票研究首次覆盖报告的结构。在构建最终报告文档时，请将此作为指南。

**注意：** 实际报告必须使用DOCX技能创建。不要生成markdown内容。

**关键要求：**
1. **生成20-30+张图表图像**：在创建Word文档之前，使用Python（matplotlib/plotly）生成图表
2. **使用DOCX技能**：创建具有适当样式、页眉/页脚和格式的专业报告
3. **嵌入实际图表图像**：将生成的PNG/JPG图表文件插入Word文档的适当位置
4. **不要使用MARKDOWN**：不要生成markdown内容。使用DOCX技能创建.docx文件。

**关键格式指导：**
- **最大密度**：每页应充满信息。在全文中穿插文本、图表和表格。
- **不要孤立章节**：永远不要让章节标题单独出现，也不要让单个图表单独占一页。
- **20-30+张实际图表图像**：将图表生成为图像文件，然后使用DOCX技能嵌入文本章节中。

## 第1页：投资更新（最重要的页面）

**关键**：第1页不是传统的执行摘要。它是**投资更新**，具有专业股票研究公司使用的特定机构格式。

**重要结构说明：**
- 这是"投资更新"或"公司更新"页面，不是"执行摘要"
- 在左上角使用评级框
- 突出显示股价表现图表（图表1）
- 包含3-4个使用■字符的详细要点
- 每个要点有**粗体主题标题**，后跟3-5句解释
- 底部有财务和估值指标表
- 所有图表必须有图表编号（图表1、图表2等）和来源行

## 图表编号和格式标准

**关键**：所有图表、图形和表格必须遵循专业股票研究中使用的严格图表编号约定。

### 图表编号格式

**每个图表/表格必须有：**
1. **连续编号**：图表1、图表2、图表3等（在整个报告中连续编号）
2. **描述性标题**："图表X - [公司] [具体指标] [图表/分析类型]"
3. **来源行**（始终在底部）："来源：公司数据，[机构名称]估计。"

**示例：**
- 图表1 - [公司]股价表现
- 图表2 - [公司]历史和预测收入按产品组合
- 图表3 - [公司]按地理区域的收入
- 图表4 - [产品名称]收入和每位患者每年的价格
- 图表5 - [公司]毛利率演变
- 图表6 - DCF敏感性分析（美元/股）
- 图表7 - 估值足球场

### 标题格式

```
图表X - [描述性标题]
[图表/表格/图形内容]
来源：公司数据，[机构名称]估计。
```

对于有多个数据来源的表格：
```
图表X - [描述性标题]
[表格内容]
来源：公司文件，FactSet，[机构名称]估计。
```

### 放置指南

- 图表应按在报告中出现的顺序编号
- 第一个图表（图表1）通常是第1页的股价图表或收入增长轨迹
- 每个图表必须在其标题正下方
- 来源行应使用较小字体、斜体，位于图表的最底部

## 使用本模板的说明

1. **第1页至关重要：** 第1页的执行摘要必须包含所有关键信息 - 快速事实、财务快照、3张图表、估值摘要、论点和风险。这是最重要的页面。

2. **最大密度：** 专业股票研究信息极其密集。每页应充满穿插的文本、图表和表格。目标是60-80%的页面覆盖率，最小化空白。

3. **不要孤立章节：** 永远不要让章节标题单独出现，或单个图表/表格单独占一页。始终组合元素。示例：不要将"财务快照"单独放在第6页 - 将其与周围内容整合。

4. **20-30+张图表：** 在全文中包含大量图形，特别强调：
   - **按产品/细分的收入**（堆叠面积图）
   - **按地理区域的收入**（堆叠条形图）
   - **财务表现趋势**（多张图表）
   - 图表应嵌入文本中，不要单独分组

5. **使用DOC技能：** 此大纲应使用DOC技能转换为专业Word文档，具有适当的格式、样式、页眉/页脚和页码

6. **穿插内容：** 文本段落应内联嵌入图表。每页应有2-4个不同的元素（表格、图表、文本块）。

7. **一致的格式：** 全文使用一致的标题、正文文本、表格和图表样式。选择颜色方案并坚持使用。

8. **引用：** 为所有数据点包含引用和来源

9. **校对：** 始终校对准确性，特别是财务数据和计算

10. **执行摘要最后写：** 虽然它出现在第1页，但在完成完整分析后最后撰写此章节

11. **平衡：** 客观地呈现正面和负面方面

12. **具体 > 泛泛：** 使用具体数据和示例，而不是泛泛的陈述
