# 首次覆盖报告质量控制检查清单

在交付首次覆盖报告之前，验证以下所有项目是否完成。

## 关键最低要求 - 报告必须满足这些

**关键：如果出现以下情况，不要交付：**
- ❌ DOCX报告少于30页 → 不完整
- ❌ 嵌入图表少于25张 → 不完整
- ❌ 综合表格少于12个 → 不完整
- ❌ 字数少于10,000字 → 不完整
- ❌ 没有XLS财务模型 → 缺失交付成果
- ❌ 图表是文本描述，而非实际的PNG/JPG文件 → 重大失败

## 交付成果检查清单

- [ ] 已创建DOCX报告文件
- [ ] 已创建XLS财务模型文件
- [ ] 两个文件命名正确：`[公司]_首次覆盖报告_[日期].docx` 和 `[公司]_财务模型_[日期].xlsx`

## DOCX报告 - 长度和内容

**长度验证：**
- [ ] 报告为30-50页（计算最终文档的页数）
- [ ] 字数为10,000-15,000字
- [ ] 如果少于30页：停止并添加更多内容

**视觉元素：**
- [ ] 嵌入25-35张图表（计算数量：_____张图表）
- [ ] 所有图表都是实际的PNG/JPG图像文件（不是文本描述）
- [ ] 包含12-20个综合表格（计算数量：_____个表格）
- [ ] 图表和表格穿插在全文中，而不是集中在末尾

**图表要求：**
- [ ] 按产品收入图表：堆叠面积格式 ✓
- [ ] 按地理收入图表：堆叠条形格式 ✓
- [ ] DCF敏感性：带颜色编码的双向热力图 ✓
- [ ] 估值足球场：水平条形图 ✓
- [ ] 所有其他图表都是实际的图像文件 ✓

**表格要求：**
- [ ] 完整利润表（40-50行），包含5年历史数据 + 5年预测
- [ ] 完整现金流量表（30-40行）
- [ ] 完整资产负债表（35-45行）
- [ ] 按产品收入表（20-30行）
- [ ] 按地理收入表（15-20行）
- [ ] 按渠道收入表（10-15行）
- [ ] 可比公司表，包含统计摘要（最大值/75分位数/中位数/25分位数/最小值）
- [ ] DCF计算表（30-40行）
- [ ] WACC计算表（8-10行）
- [ ] 两个敏感性表格
- [ ] 2-3个额外的财务/竞争表格

## DOCX报告 - 结构

**第1页要求：**
- [ ] 存在"首次覆盖"标题（不是"公司更新"）
- [ ] 以论点为中心的标题（不是事件驱动的标题，如"强劲的第四季度业绩"）
- [ ] 评级框，包含评级、价格、目标价、52周区间、市值、企业价值
- [ ] 3-4个段落长度的要点，使用■字符和粗体标题
- [ ] 财务和估值指标表，包含2-3年历史数据和2年预测
- [ ] 表格显示实际数据用"A"后缀，估计数据用"E"后缀
- [ ] 所有视觉元素都有来源行

**内容章节：**
- [ ] 目录（第2页）
- [ ] 投资论点和风险（3-5页）
- [ ] 公司概述（6-12页），包括：
  - [ ] 公司描述
  - [ ] 历史和里程碑
  - [ ] 管理层简介（每位高管300-400字，共3-4位高管）
  - [ ] 产品/服务详情
  - [ ] 竞争格局
- [ ] 财务分析和预测（10-15页）
- [ ] 估值分析（8-12页）
- [ ] 假设章节（2,000-3,000字，记录所有预测假设）
- [ ] 情景分析（1,500-2,000字，包含牛市/基准/熊市参数）
- [ ] 附录，包括数据来源和参考页面

## DOCX报告 - 格式

**图表和表格格式：**
- [ ] 每个图表上方有标题："图表X - [公司] [描述性标题]"
- [ ] 每个图表下方有来源行："来源：[具体来源及日期]"
- [ ] 图表编号连续（图表1、2、3...无间隔）
- [ ] 每个表格有带阴影的标题行
- [ ] 每个表格底部有来源行
- [ ] 所有年份使用"A"表示实际数据，"E"表示估计数据

**专业格式：**
- [ ] 全文使用一致的字体（Calibri、Arial或类似字体）
- [ ] 页眉和页脚包含页码
- [ ] 紧凑布局：60-80%页面覆盖率，最小化空白
- [ ] 每页同时包含文本和视觉元素（图表或表格）
- [ ] 使用专业商业报告模板

## 引用和来源 ⭐⭐⭐ 关键

**来源归属：**
- [ ] 每个图表有具体来源，包含文档名称和日期
- [ ] 每个表格有具体来源，包含文档引用
- [ ] 文中的关键统计数据有带来源的脚注
- [ ] 不是泛泛的"公司数据" - 必须具体

**超链接：** ⭐⭐⭐ 强制要求
- [ ] 所有URL都是可点击的超链接（不是纯文本）
- [ ] SEC文件超链接到EDGAR查看器
- [ ] 财报电话会议超链接（Seeking Alpha或公司投资者关系）
- [ ] 新闻稿超链接到公司投资者关系页面
- [ ] 演示文稿超链接到PDF URL
- [ ] 行业报告超链接（如公开可用）
- [ ] 订阅数据（彭博、FactSet）注明"（需要订阅）"
- [ ] 不显示任何原始URL - 全部格式化为超链接
- [ ] 测试3-5个示例超链接以确保它们有效（Ctrl+点击）

**参考页面：**
- [ ] 报告末尾有"数据来源和参考"页面
- [ ] 列出报告中使用的所有来源
- [ ] 来源按类别组织（SEC文件、财报电话会议等）
- [ ] 每个来源都有日期
- [ ] 每个来源都有可点击的超链接（如适用）

## XLS财务模型 - 结构

**文件结构：**
- [ ] Excel工作簿包含15+个标签页
- [ ] 标签页包括：执行摘要、假设、历史财务数据、收入模型、营业费用、利润表、资产负债表、现金流量表、辅助附表、DCF估值、可比公司分析、先例交易、情景分析、敏感性分析、图表

**格式：**
- [ ] 硬编码输入使用蓝色文本
- [ ] 公式使用黑色文本
- [ ] 链接到其他工作表使用绿色文本
- [ ] 专业格式，包含边框和阴影
- [ ] 清晰的章节标题和标签

**模型功能：**
- [ ] 所有数字流动（更改假设 → 整个模型更新）
- [ ] DCF链接到假设和预测
- [ ] 无循环引用或错误
- [ ] 所有重要单元格/范围都已命名
- [ ] 敏感性表格动态工作

## XLS财务模型 - 内容

**预测：**
- [ ] 3-5年历史数据
- [ ] 5年前瞻预测（FY+1至FY+5）
- [ ] 收入按产品、地理、渠道细分
- [ ] 完整损益表，包含40-50行项目
- [ ] 完整现金流量表，包含30-40行项目
- [ ] 完整资产负债表，包含35-45行项目

**估值：**
- [ ] 完整的DCF模型，显示所有计算
- [ ] WACC计算，包含所有组成部分
- [ ] 终值计算
- [ ] 可比公司分析（5-10家公司）
- [ ] 先例交易分析（5-10笔交易）
- [ ] 情景分析（牛市/基准/熊市）
- [ ] 两个敏感性表格

## 跨文件一致性

**关键**：DOCX和XLS之间的数字必须完全匹配

- [ ] 收入数字在两个文件中匹配
- [ ] EPS数字在两个文件中匹配
- [ ] 利润率百分比在两个文件中匹配
- [ ] 估值数字在两个文件中匹配
- [ ] 目标价在两个文件中匹配
- [ ] 所有预测年份在两个文件中匹配

**验证方法**：抽查DOCX报告和XLS模型之间的10-15个关键数字。

## 内容质量

**投资论点：**
- [ ] 3-5个清晰的论点支柱
- [ ] 每个支柱有具体数据和量化支持
- [ ] 每个支柱的财务影响已量化
- [ ] 已识别催化剂及时间线

**分析深度：**
- [ ] 全面的商业模式分析
- [ ] 详细的竞争评估
- [ ] 分析3-5年财务趋势
- [ ] 识别并量化8-12个风险
- [ ] 管理团队分析（每位高管300-400字）

**假设：**
- [ ] 2,000-3,000字记录所有假设
- [ ] 按类别/地理的收入增长假设
- [ ] 利润率假设，包含显示驱动因素的桥梁
- [ ] 营运资本假设
- [ ] 资本支出假设
- [ ] 每个假设都有具体的量化

**情景：**
- [ ] 1,500-2,000字的情景分析
- [ ] 牛市情景，包含具体参数和催化剂
- [ ] 基准情景，包含详细理由
- [ ] 熊市情景，包含具体触发因素
- [ ] 每个情景的概率评估

## 写作质量

**风格：**
- [ ] 以数字开头（"收入增长15%至12亿美元"，而不是"强劲的收入"）
- [ ] 使用"vs."而不是"versus"
- [ ] 直接简洁
- [ ] 全文使用专业的机构语调
- [ ] 无非正式语言

**准确性：**
- [ ] 股票代码无拼写错误
- [ ] 公司名称无拼写错误
- [ ] 所有日期准确
- [ ] 所有计算已验证
- [ ] 图表与文本描述匹配
- [ ] 所有数字格式正确（美元符号、百分号、逗号）

## 交付前最终检查

快速完成以下最终审查：

1. **交付成果**：已创建DOCX和XLS文件 ✓
2. **长度**：DOCX为30-50页 ✓
3. **图表**：嵌入25-35个实际的PNG/JPG文件 ✓
4. **表格**：包含12-20个综合表格 ✓
5. **字数**：10,000-15,000字 ✓
6. **超链接**：测试3-5个超链接 - 全部有效 ✓
7. **交叉检查**：抽查10个数字在DOCX和XLS之间匹配 ✓
8. **第1页**：存在"首次覆盖"标题 ✓

如果任何项目失败，不要交付。返回并修复。
