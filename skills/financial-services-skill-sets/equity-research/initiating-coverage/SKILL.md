# 首次覆盖

通过结构化的5任务工作流程创建机构级股票研究首次覆盖报告。每个任务必须单独执行并验证输入。

## 概述

本技能按照国内机构标准（中金、华泰、海通等格式）制作全面的首次覆盖报告，适用于A股、港股及中概股上市公司。任务单独执行，每个任务在继续之前验证前置条件。

**默认字体**：所有文档使用思源宋体或用户指定的字体（除非用户另有说明）。

## 任务1：公司研究

**目的**：研究公司的业务、管理层、竞争地位、行业和风险。

**前置条件**：✅ 无（完全独立）
- 公司名称或股票代码

**流程**：
1. 验证提供的公司名称/股票代码
2. 从references/task1-company-research.md加载详细说明
3. 执行定性研究工作流程
4. 交付研究文档

**输出**：公司研究文档（6,000-8,000字）
- 公司概述和历史
- 管理层简介（300-400字 × 3-4位高管）
- 产品和服务分析
- 行业概述
- 竞争分析（5-10个竞争对手）
- TAM规模测算
- 风险评估（8-12个风险）

**文件名**：`[公司]_研究文档_[日期].md`

**⚠️ 仅交付这1个文件。不要完成摘要，不要额外文档。**

**⚠️ 不要走捷径：**
- ✅ 撰写完整的6,000-8,000字（不是摘要）
- ✅ 为所有3-4位高管完成300-400字的简介
- ✅ 彻底分析所有5-10个竞争对手
- ✅ 覆盖所有4个类别的8-12个风险
- ❌ 不要为了节省时间而缩写章节
- ❌ 不要跳过任何必需的章节

**继续之前的验证**：此任务无需验证。

## 任务2：财务建模

**目的**：提取历史财务数据并构建包含预测和情景的综合Excel财务模型。

**前置条件**：⚠️ 开始前验证
- **必需**：访问公司财务数据
  - 对于上市公司：上交所/深交所/港交所的最新年报/半年报
  - 对于私营公司：财务报表或可用估计
  - 或：用户提供预先提取的历史财务数据
- **可选**：公司研究（任务1）用于业务背景

**输入验证**：
```
开始前 - 选择方法：

选项A：提取财务数据（最常见）
- [ ] 有权访问年报/半年报或财务报表？
- [ ] 准备提取3-5年的数据？

选项B：用户提供预先提取的财务数据
- [ ] 收到历史财务数据文件？
- [ ] 包含利润表、现金流量表、资产负债表（3-5年）？

可选：
- [ ] 公司研究（任务1）完成以提供背景？
```

**流程**：
1. 验证财务数据访问
2. 从references/task2-financial-modeling.md加载详细说明
3. **步骤1**：提取历史财务数据（如需要）
4. **步骤2+**：构建包含6个基本标签页的预测模型
5. 交付Excel模型

**输出**：Excel财务模型（.xlsx）
- 6个基本标签页：
  1. **收入模型** - 产品细分（20-30行）+ 地理细分（15-20行）
  2. **利润表** - 完整损益表，40-50行项目，历史（3-5年）+ 预测（5年）
  3. **现金流量表** - 经营/投资/筹资活动，历史 + 预测
  4. **资产负债表** - 资产/负债/权益，历史 + 预测
  5. **情景分析** - 牛市/基准/熊市比较表
  6. **DCF输入** - 为任务3估值准备

**文件名**：`[公司]_财务模型_[日期].xlsx`

**⚠️ 仅交付这1个文件。不要完成摘要，不要额外文档。**

**⚠️ 不要走捷径：**
- ✅ 如果提取财务数据：从3份财务报表中提取所有行项目（3-5年）
- ✅ 完整构建所有6个预测标签页，包含完整细节
- ✅ 创建详细的收入模型，包含20-30个产品行和15-20个地理行
- ✅ 构建完整的利润表，包含40-50行项目（不是缩写版）
- ✅ 包含完整的现金流量表和资产负债表，包含所有行项目
- ✅ 完成所有三种情景（牛市/基准/熊市），使用不同参数
- ❌ 不要创建简化/缩写版本
- ❌ 不要跳过任何6个基本标签页
- ❌ 不要跳过历史财务数据提取（如需要）

**继续任务3之前的验证**：
- [ ] 历史财务数据已提取（如需要）或已提供
- [ ] Excel文件已创建并可打开
- [ ] 模型包含所有6个基本标签页（收入模型、利润表、现金流量表、资产负债表、情景分析、DCF输入）
- [ ] 历史数据（3-5年）已纳入
- [ ] 预测已完成（未来5年）
- [ ] 情景已完成（牛市/基准/熊市）
