# 催化剂日历

## 工作流程

### 步骤1：定义覆盖范围

- 需跟踪的公司列表（股票代码或名称）
- 行业/板块关注
- 是否包含宏观事件？（美联储会议、经济数据、监管截止日期）
- 时间范围（未来2周、1个月、1个季度）

### 步骤2：收集催化剂

为每家公司识别即将发生的事件：

**财报与金融事件**
- 季度财报日期和时间（盘前/盘后）
- 年度股东大会
- 投资者日/分析师日
- 资本市场日
- 债务到期/再融资日期

**公司事件**
- 产品发布或公告
- FDA批准/监管决策
- 合同续约或到期
- 并购里程碑（完成日期、监管批准）
- 管理层变动
- 内部人交易窗口（锁定期到期）

**行业事件**
- 主要会议（日期、参会演讲公司）
- 贸易展会和博览会
- 监管意见期或裁决
- 行业数据发布（月度销售、流量等）

**宏观事件**
- 美联储会议（FOMC日期）
- 就业报告、CPI、GDP发布
- 央行决策（欧洲央行、日本央行等）
- 对市场有影响的地缘政治事件

### 步骤3：日历视图

| 日期 | 事件 | 公司/行业 | 类型 | 影响（高/中/低） | 我们的仓位 | 备注 |
|------|------|-----------|------|------------------|------------|------|
|      |      |           | 财报/公司/行业/宏观 |            | 多头/空头/中性 |      |

### 步骤4：每周预览

每周生成前瞻性摘要：

**本周关键事件：**
1. [日期]：[公司]第[X]季度财报 —— 共识预期[X美元每股收益]，我们的预期[X美元]，重点关注：[指标]
2. [日期]：[事件] —— 对[股票]的重要性
3. [日期]：[宏观数据发布] —— 预期和仓位策略

**下周预览：**
- 对即将到来的重要事件的提前提醒

**仓位影响：**
- 可能影响特定仓位的事件
- 建议的预先仓位部署
- 二元事件前的风险管理

### 步骤5：输出

- 带有日历视图和可排序列的Excel工作簿
- 每周预览邮件/说明（Markdown格式）
- 可选：与Google日历集成

## 重要注意事项

- 财报日期可能变动 —— 临近日期时需通过公司投资者关系页面和彭博/万得进行核实
- 预公告风险：跟踪有预公告历史的公司（正面或负面）
- 会议出席名单很有价值 —— 哪些公司参会演讲，哪些公司明显缺席？
- 一些催化剂是定期发生的（月度行业数据）—— 建立模板并自动填充
- 按影响程度进行颜色编码：红色=高影响，黄色=中等影响，绿色=常规
- 归档过去的催化剂及其实际结果 —— 随着时间推移建立模式识别能力
