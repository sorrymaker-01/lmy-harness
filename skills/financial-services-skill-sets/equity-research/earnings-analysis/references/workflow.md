# 盈利更新的详细工作流程

本文档提供盈利更新过程每个阶段的详细分步说明。

## ⚠️⚠️⚠️ 重要警告：始终使用最新的盈利数据 ⚠️⚠️⚠️

**请先阅读此内容：**

训练数据已过时。积极搜索并检索最新的盈利材料。使用过时的盈利数据是盈利分析中最常见的错误。

**开始前：**
1. **检查今天的日期** - 写下当前日期
2. **搜索最新信息** - 使用网络搜索查找最新的盈利信息
3. **验证日期** - 确认盈利发布在过去3个月内
4. **如果超过3个月** - 获得了错误的季度，重新搜索

## 阶段1：盈利数据收集（30-60分钟）

### 步骤1：确定最新的盈利期间

**重要**：始终搜索最新的盈利信息 - 不要依赖知识截止日期。
**重要**：永远不要使用训练中的盈利数据 - 它已过时。

**步骤1a：搜索最新的盈利发布**

**🚨 积极搜索 - 训练数据已过时。 🚨**

**必须步骤1：检查今天的日期**
- **明确写下今天的日期**：[月] [日]，[年]
- **使用此日期验证** 找到的任何盈利信息是否在3个月内
- **示例**："今天是2024年10月29日"

**必须步骤2：搜索"最新盈利"**
- **使用网络搜索**，查询如下：
  - `[公司名称] latest earnings results`
  - `[公司名称] most recent quarterly earnings`
  - `[股票代码] earnings latest quarter`
- **或搜索公司投资者关系网站**：
  - 访问 `investor.[公司].com` 或 `[公司].com/investors`
  - 导航至"新闻稿"、"新闻"或"盈利"部分
  - **按日期排序以找到最新发布**
  - 查找关键词："earnings"、"results"、"financial results"、"quarterly results"

**必须步骤3：验证发布日期**
- **查看找到的盈利发布日期**
- **计算**：此日期是否在今天的过去3个月内？
- **如果是** → 继续下一步
- **如果否（超过3个月）** → 🚨 错误的季度 - 重新搜索更近期的信息

**❌ 需避免的常见错误：**
- ❌ 不搜索就使用训练中的盈利数据
- ❌ 基于预期假设"2024年第三季度"是最新的
- ❌ 不检查日期就抓取找到的第一个盈利发布
- ❌ 不将发布日期与今天的日期进行比较
- ❌ 当发布已超过4个月时继续进行

**✅ 正确方法：**
- ✅ 首先检查今天的日期
- ✅ 明确搜索"latest"或"most recent"
- ✅ 阅读材料上的实际发布日期
- ✅ 确认发布日期在今天的3个月内
- ✅ 如有疑问，使用不同术语重新搜索

**必须步骤4：确定季度**
- **阅读标题/头条**以确定季度（Q1、Q2、Q3、Q4或财季）
- **阅读文档本身的发布日期**
- **验证季度名称和日期都是近期的**

3. **如果IR网站不明确，替代搜索方法：**
   - 网络搜索：`[公司名称] latest earnings results`
   - 网络搜索：`[公司名称] most recent quarterly earnings`
   - 网络搜索：`[股票代码] earnings latest quarter`
   - 上交所/深交所/港交所公告：搜索公司并查看最新的年报/半年报/季报公告日期

**找到最新数据的示例搜索：**
- "Nike latest earnings results" → 返回最新报告的季度
- "AAPL most recent quarterly earnings" → 显示苹果最新盈利
- "Tesla Q3 2024 earnings" → 结果确认2024年第三季度存在

**步骤1b：了解公司的财年日历**

从搜索中确定最新季度后，了解公司的财年以正确解释：

**常见财年模式：**
- **日历年（CY）**：Q1=1-3月，Q2=4-6月，Q3=7-9月，Q4=10-12月
- **耐克财年**：Q1=6-8月，Q2=9-11月，Q3=12-2月，Q4=3-5月（5月财年结束）
- **苹果财年**：Q1=10-12月，Q2=1-3月，Q3=4-6月，Q4=7-9月（9月财年结束）
- **沃尔玛财年**：Q1=2-4月，Q2=5-7月，Q3=8-10月，Q4=11-1月（1月财年结束）

许多公司在盈利发布标题中说明其财年。如有需要，搜索`[公司] fiscal year calendar`。

**步骤1c：必须验证 - 验证获得的最新数据**

🛑 **停止 - 在验证以下所有项目之前不要继续：**

- [ ] ✅ **今天的日期已写下**：[月] [日]，[年]
- [ ] ✅ **积极搜索** 使用"latest earnings"或"most recent earnings"
- [ ] ✅ **找到的盈利发布日期**：[月] [日]，[年]
- [ ] ✅ **验证发布在今天的3个月内**（计算一下！）
- [ ] ✅ **未仅基于今天的日期假设** 季度
- [ ] ✅ **能看到确认季度/期间的实际新闻稿**
- [ ] ✅ **已打开并阅读** 实际的盈利材料（不仅仅是假设它们存在）

**🚨 红旗 - 如果以下任何情况为真，获得了错误的季度：**
- 🚨 发布日期超过90天
- 🚨 依赖预期而非通过搜索找到的内容
- 🚨 实际上没有看到确认该季度存在的新闻稿或文件
- 🚨 使用了训练中的数据而没有搜索
- 🚨 无法说明确切的发布日期
- 🚨 找到的发布日期来自2023年或更早（当今天是2024年+时）

**如果存在任何红旗**：停止并重新搜索。不要使用过时数据继续。

**步骤1c：处理命名变体**

公司使用不同的术语 - 识别这些模式：

**季度术语：**
- "Q1 2024"、"Q1 FY24"、"First Quarter 2024"、"1Q24"
- "Third Quarter Fiscal 2024"、"Q3 FY2024"、"3Q FY24"

**盈利发布标题：**
- "[公司] Reports Q3 2024 Results"
- "[公司] Announces Third Quarter Fiscal 2024 Financial Results"
- "[公司] Q3 Revenue Grew 15% Year-over-Year"

**国内财报文件搜索：**
- 公司名称可能与常用名称不同（例如，"阿里巴巴集团控股有限公司" vs "阿里巴巴"）
- 通过股票代码搜索公告更可靠
- 查找最新的年报（年度）、半年报（半年度）、季报（季度）

### 步骤2：收集盈利材料

在搜索并确认最新季度后，收集以下内容：

**⚠️ 重要：搜索并访问实际文档 - 不要依赖训练数据。**

**主要材料（必需）：**
- **盈利新闻稿** - 通常在公司投资者关系网站的"新闻稿"或"新闻"下
  - 导航至IR网站并找到实际新闻稿
  - 搜索模式："[公司名称] latest earnings"、"[公司名称] Q[X] [年] earnings results"
  - 查找PDF或HTML版本
  - **验证日期与步骤1中找到的日期匹配**（应在过去1-3个月内）
  - **阅读实际文档**以确认季度并获取报告数字

- **国内财务报告** - 在上交所/深交所/港交所披露易
  - 通过股票代码搜索
  - 对于第1-3季度：查找最新的季报
  - 对于年度：查找年报（年度报告）
  - 注意：可能在业绩预告/快报发布后1-5天正式披露
  - 直接链接格式：`http://www.cninfo.com.cn/new/disclosure/detail?plate=szse&orgId=...` 或 `https://www.hkexnews.hk/`

- **业绩说明会记录** - 🚨 **验证记录上的日期** 🚨
  - **搜索**："[公司] 最新业绩说明会"或"[公司] Q[X] [年] 业绩说明会"
  - **来源**：
    - 公司IR网站（有些直接发布记录）
    - 同花顺/东方财富：搜索"[公司] [最新季度] 业绩说明会"
    - 路演中、进门财经（替代来源）
  - **关键日期检查**：
    - ✅ **使用任何记录前，验证记录本身的日期**
    - ✅ **记录日期必须与步骤1中的业绩发布日期匹配**
    - ✅ **如果记录显示"2023年Q2"但发布是"2024年Q3"，则获得了错误的记录**
    - 🚨 **常见错误**：不检查日期就抓取旧记录
  - 如果记录尚未可用，收看网络直播回放或注明等待记录

**补充材料（如有）：**
- **投资者演示/幻灯片** - 通常与新闻稿一起发布在IR网站
  - 通常标题为"Q[X] [年] Earnings Presentation"或"Investor Presentation"
  - PDF格式，包含管理层在盈利电话会议期间展示的幻灯片

- **补充数据文件** - 一些公司提供包含详细指标的Excel文件
  - 查找"Supplemental Financial Information"或"Investor Data Sheet"

**参考材料（用于比较）：**
- **上一季度业绩** - 用于环比比较
  - 来自上一季度的盈利发布（90天前）

- **去年同期** - 用于同比比较
  - 来自去年同一季度（4个季度前）

- **先前估计** - 如果该公司之前被覆盖
  - 来自上次盈利更新或首次覆盖报告
  - 检查本季度指标的先前估计

- **共识估计** - 来自Wind、TuShare、同花顺、东方财富、Choice
  - 重要：使用业绩预告/快报发布前的估计
  - 查找"截至[预告前日期]"以确保是发布前的共识
  - 用于超出/低于预期分析

**🛑 继续步骤3前的必须验证：**

**日期 - 验证所有日期匹配：**
- [ ] ✅ **今天的日期已写下**：_______________
- [ ] ✅ **盈利发布日期**：_______________（必须在今天的3个月内）
- [ ] ✅ **盈利电话会议记录日期**：_______________（必须与发布日期±1天匹配）
- [ ] ✅ **国内财务报告日期**：_______________（必须与发布同一季度）
- [ ] ✅ **所有材料显示同一季度**（例如，都显示"2024年Q3"，而非混合季度）

**搜索与访问 - 验证完成积极搜索：**
- [ ] ✅ **搜索**了"latest earnings"（非基于当前日期假设）
- [ ] ✅ **访问**了实际的盈利新闻稿并阅读
- [ ] ✅ **打开**了实际的盈利电话会议记录并验证日期
- [ ] ✅ **确认**这是通过检查日期确认的最新季度
- [ ] ✅ 从实际发布中获得完整的财务业绩（收入、每股收益、利润率等）
- [ ] ✅ 有带来源日期的盈利前共识估计

**🚨 红旗 - 如果以下任何情况为真，停止：**
- 🚨 实际上没有搜索或访问盈利材料
- 🚨 基于记忆或训练数据而非当前文档工作
- 🚨 盈利发布日期超过90天
- 🚨 无法说明盈利发布的确切日期
- 🚨 记录日期与发布日期不匹配
- 🚨 材料显示不同季度（例如，发布说Q3但记录说Q2）
- 🚨 不验证日期就抓取第一个结果

### 步骤4：从电话会议中识别关键主题

收听或阅读盈利电话会议记录并注意：
- 管理层的语气（自信、谨慎、防御性？）
- 强调的关键主题（产品发布、地理趋势、竞争）
- 分析师的问题（投资者关注什么？）
- 提供的指引（上调、下调、维持、新提供？）
- 任何意外或未预期的评论

## 阶段2：分析（2-3小时）

### 步骤5：超出/低于预期分析

对于每个超出或低于预期的关键指标，解释：

**如果超出预期：**
- 什么驱动了超出？
- 是一次性还是可持续的？
- 管理层是否上调了未来指引？
- 这如何影响我们的论点？

**如果低于预期：**
- 出了什么问题？
- 是公司特定还是行业范围的？
- 管理层是否采取纠正措施？
- 这如何影响我们的论点？

**示例格式：**
```
■ **收入超出3%，由强劲的DTC表现驱动**

收入135亿美元超出我们131亿美元的估计4亿美元（3%），超出共识132亿美元3亿美元（2%）。超出主要由直接面向消费者渠道驱动，该渠道同比增长18%（vs.我们12%的估计），抵消了弱于预期的批发业务（-5% vs.持平估计）。管理层引用强劲的数字需求和成功的产品发布（Pegasus 40跑鞋、新的Jordan配色）作为关键驱动因素。DTC现在占总收入的42%，而一年前为38%，展示了成功的渠道转变策略。
```

### 步骤6：细分/地理/产品分析

按以下方面分析表现：
- 业务细分（如果是多细分公司）
- 地理（北美、欧洲、中国等）
- 产品类别
- 渠道（零售、批发、电子商务）

识别：
- 什么超出预期？
- 什么低于预期？
- 与前几个季度的趋势
- 管理层对每个领域前景的评论

### 步骤7：利润率分析

分析盈利能力：
- 毛利率：上升还是下降？为什么？
- 营业利润率：上升还是下降？为什么？
- 关键驱动因素（定价、组合、成本、杠杆）
- 未来展望

### 步骤8：指引分析

如果公司提供指引：
- 将新指引与先前指引比较
- 与内部估计和华尔街估计比较
- 评估可信度（公司有保守指引的历史吗？超出？）
- 识别指引背后的关键假设

如果公司未提供指引：
- 明确注明
- 基于业绩和评论提供独立展望

### 步骤9：更新财务模型

更新以下估计：
- 当前年度（剩余季度）
- 下一年度
- 可能的后一年度

**清晰显示：**
```
更新的估计：
─────────────────────────────────────────────────
                        旧估计     新估计     变化      原因
2024财年收入         $XX.XB      $XX.XB      +X.X%      [简要原因]
2024财年EBITDA          $X.XB       $X.XB       +X.X%      [简要原因]
2024财年每股收益             $X.XX       $X.XX       +X.X%      [简要原因]

2025财年收入         $XX.XB      $XX.XB      +X.X%      [简要原因]
2025财年EBITDA          $X.XB       $X.XB       +X.X%      [简要原因]
2025财年每股收益             $X.XX       $X.XX       +X.X%      [简要原因]
```

### 步骤10：更新估值与目标价

基于更新的估计：
- 重新计算DCF（使用更新的现金流）
- 更新可比公司倍数（如果同行已报告）
- 确定新的公允价值
- 决定目标价是否变化

**目标价决策：**
- 如果估计变化显著（>5%）→ 通常更改目标价
- 如果估计变化微小（<5%）→ 可能维持目标价
- 如果论点加强/削弱 → 即使没有估计变化也可能更改

### 步骤11：评估评级影响

决定是否更改评级：
- 如果业绩显著好于预期 + 指引上调 → 考虑升级
- 如果业绩显著差于预期 + 指引下调 → 考虑降级
- 如果符合预期或喜忧参半 → 通常维持评级

**考虑：**
- 股票反应（上涨/下跌/持平？）
- 估值（相对于新估计贵/便宜？）
- 风险/回报（不对称是否改变？）

## 阶段3：图表生成（1-2小时）

### 步骤12：生成8-12个图表

创建关注季度趋势和新内容的图表。

**必需图表（共8-12个）：**

1. **季度收入进展**（柱状图）
   - 最近8-12个季度
   - 显示每个季度与估计的超出/低于
   - 突出当前季度

2. **季度每股收益进展**（柱状图）
   - 最近8-12个季度
   - 显示与估计的超出/低于
   - 调整后和GAAP

3. **季度利润率趋势**（折线图）
   - 毛利率、EBIT利润率、净利润率
   - 最近8-12个季度
   - 显示轨迹

4. **按细分/地理划分的收入**（堆积柱状图或表格）
   - 当前季度 vs. 同比
   - 按细分的增长率

5. **关键运营指标**（多折线图）
   - 客户数量、ARPU、销售单位等（相关指标）
   - 最近8-12个季度

6. **超出/低于预期摘要**（瀑布图或表格）
   - 显示超出/低于的组成部分
   - 什么驱动了与估计的差异

7. **估计修订图表**（前后比较）
   - 旧财年估计 vs. 新财年估计
   - 显示变化的柱状图

8. **估值图表**（市盈率或EV/EBITDA倍数）
   - 历史倍数范围
   - 当前倍数
   - 公允价值倍数

**可选图表（如果空间允许）：**
- 同行比较（如果同行已报告）
- 指引 vs. 华尔街比较
- 现金流指标
- 资产负债表亮点（如有显著变化）

**图表风格指南：**
- 关注趋势（季度进展）
- 突出变化（超出/低于预期，估计修订）
- 保持简单清晰（这是快速周转报告）

## 阶段4：报告创建（2-3小时）

### 步骤13：创建DOCX报告

使用DOCX技能创建8-12页报告。

请参阅[report-structure.md](report-structure.md)了解完整的逐页模板和格式要求。

**关键步骤：**
1. 创建第1页，包含盈利摘要和快速要点
