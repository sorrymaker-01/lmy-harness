# 报告结构和模板

本文档提供盈利更新DOCX报告的完整逐页模板和格式要求。

## 完整报告结构

**报告结构：**

---

## 第1页：盈利摘要

**顶部部分 - 标题：**
```
[公司名称] ([股票代码])
[季度] [年份] 盈利更新

[当前日期]

评级：[维持/上调/下调] [评级]
价格（截至[日期]）：$XX.XX
目标价：[如有变化则旧 → 新，或维持 $XXX]
```

**顶部部分 - 快速摘要框：**
```
盈利摘要
─────────────────────────────────────────────────
第[X]季度 [年份] 业绩：[超出预期 / 符合预期 / 低于预期]

                报告值    估计值    差异
收入             $X,XXX      $X,XXX   +$XXX (+X%)
每股收益（调整后） $X.XX       $X.XX    +$X.XX (+X%)

关键要点：
■ [要点1 - 一句话]
■ [要点2 - 一句话]
■ [要点3 - 一句话]
```

**主要内容 - 投资影响（3-4个项目符号）：**

使用■字符，配合**粗体标题**和段落长度的解释：

```
■ **业绩因[细分/地理/产品]强劲而超出预期，保持积极势头**

第[X]季度收入$X.XB超出我们$X.XB的估计X%，超出共识预期X%，
主要由[具体驱动因素]驱动。[细分]收入同比增长X%（vs.我们的
X%估计），而[细分]增长X%（vs.X%估计）。管理层强调
[具体产品/计划]是关键增长驱动因素，并对前景保持自信
语气。这一超出预期展示了[论点要点]，强化了我们的积极
观点。

■ **尽管存在[逆风]，利润率同比扩大XX个基点，展示运营杠杆**

[详细利润率分析段落...]

■ **指引上调/维持/下调 - 暗示[解读]**

[详细指引分析段落...]

■ **维持[评级]，[上调/不变]目标价至$XXX**

[投资结论段落...]
```

**底部部分 - 更新的估计表格：**

```
更新的财务估计
─────────────────────────────────────────────────────────────────
                     2024财年E（旧）  2024财年E（新）  变化  2025财年E（新）
收入（百万美元）         XX,XXX         XX,XXX         +X%     XX,XXX
收入增长率（%）   X.X%           X.X%           +XXbps  X.X%
毛利率（%）     XX.X%          XX.X%          +XXbps  XX.X%
EBITDA（百万美元）          X,XXX          X,XXX          +X%     X,XXX
EBITDA利润率（%）    XX.X%          XX.X%          +XXbps  XX.X%
每股收益（调整后）（美元）   X.XX           X.XX           +X%     X.XX
市盈率（倍）              XX.Xx          XX.Xx          -X%     XX.Xx

注："E" = 估计。旧估计来自[先前报告日期]。
```

- 同比和环比趋势
- 与指引的比较（如有提供）

## 格式要求

### 1. 第1页要求
- 清晰的评级（维持跑赢大盘，上调至买入等）
- 突出显示更新的目标价
- 带有旧/新估计的摘要表格
- 3-4个段落长度的项目符号，使用■字符

### 2. 所有表格要求
- 底部有来源行
- 清晰的列标题
- 标题行底纹

### 3. 所有图表要求
- 上方有"图X - [标题]"说明
- 下方有"来源：[来源]"行
- 专业样式

### 4. 年份标记
- 使用A表示实际（2024年第三季度A）
- 使用E表示估计（2024年第四季度E）

### 5. 写作风格
- 以数字开头（"收入增长15%至12亿美元"而非"强劲收入增长"）
- 使用"vs."而非"versus"
- 直接简洁
- 聚焦于新内容

### 6. 超链接要求 ⭐⭐⭐
- 所有URL必须是Word中的可点击超链接
- 蓝色、带下划线的文本，按Ctrl+点击打开
- 显示文本有意义（非原始URL）
- 每个来源引用应在适用处有可点击链接
- 无纯文本URL - 始终格式化为超链接
