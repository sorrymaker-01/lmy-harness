# 最佳实践、示例和质量指南

本文档提供示例、成功技巧、需避免的常见错误和全面的质量检查清单。

## 示例标题

### 好的盈利更新标题：
- "茅台 2024年第三季度：茅台酒量价齐升超预期 - 维持买入评级，目标价2200元"
- "腾讯 2024年第三季度：游戏业务回暖，视频号变现加速 - 维持买入评级，目标价500港元"
- "宁德时代 2024年第三季度：动力电池龙头地位稳固，储能业务高增 - 维持买入评级，目标价300元"

### 坏的标题（避免）：
- "耐克季度更新"（过于通用，无要点）
- "公司公布盈利"（陈述显而易见的事实，无分析）
- "第三季度业绩分析"（无公司名称，无观点）

## 成功技巧

1. **速度至关重要**：在盈利发布后24-48小时内发布，而不是数天后

2. **以结论开头**：是超出还是低于预期？上调还是下调估计？

3. **量化一切**："强劲"毫无意义，"12亿元人民币超出1.5亿元或3%"才清晰

4. **关注驱动因素**：不要只说"收入超出预期"，要解释原因

5. **展示分析过程**：旧估计 → 新估计，附理由

6. **如果有实质性变化，更新目标价**：如果估计变化>5%，通常目标价也应变化

7. **参考电话会议**：引用管理层评论，不要只分析新闻稿

8. **与同行比较**：如果类似公司已公布业绩，注明相对表现

9. **简洁明了**：这不是全面报告，聚焦于季度业绩

10. **图表展示趋势**：季度进展图表最有价值

## 需避免的常见错误

❌ **过于全面**：不要为季度业绩撰写首次覆盖长度的报告

❌ **缺少超出/低于预期分析**：首先说明业绩是超出还是低于预期

❌ **未更新估计**：必须提供更新后的前瞻性估计

❌ **语言模糊**："强劲表现"而不量化

❌ **忽视指引**：如果公司提供指引，要彻底分析

❌ **太慢**：盈利后5天以上发布失去相关性

❌ **重复基础知识**：不要花3页解释公司做什么

❌ **缺少目标价更新**：如果估计有实质性变化，目标价也应更新

❌ **无投资影响**：必须将业绩与投资论点和评级联系起来

❌ **缺少引用**：每个数字都需要带可点击超链接的来源

❌ **纯文本URL**：所有URL必须格式化为可点击超链接

## 全面质量控制检查清单

在交付盈利更新前，验证以下所有项目：

### 内容与分析检查清单

**超出/低于预期分析：**
- [ ] 超出/低于预期分析作为报告开头
- [ ] 具体差异已量化（例如，"超出1.2亿元人民币或3%"）
- [ ] 解释业绩与预期不同的原因
- [ ] 分析每个关键指标（收入、每股收益、利润率等）

**指标与表现：**
- [ ] 所有关键指标都有同比比较
- [ ] 相关处包含环比比较
- [ ] 提供细分/地理/产品分类
- [ ] 分析运营指标（客户、ARPU、单位等）

**指引与估计：**
- [ ] 分析并量化指引变化（如有提供）
- [ ] 如果无指引，明确注明
- [ ] 提供当前年度的更新估计
- [ ] 提供下一年度的更新估计
- [ ] 清晰显示旧估计与新估计
- [ ] 解释变化内容及原因

**估值与评级：**
- [ ] 根据业绩更新目标价（如有必要）
- [ ] 如果目标价不变，明确维持
- [ ] 解释估值方法
- [ ] 确认或更改评级，提供明确理由
- [ ] 评估投资论点，必要时更新

### 格式与长度检查清单

**整体结构：**
- [ ] 报告为8-12页（不长不短）
- [ ] 第1页有盈利摘要格式
- [ ] 第1页标题包含"盈利更新"（非"首次覆盖"）
- [ ] 事件驱动标题（例如，"强劲的第三季度业绩..."）

**表格：**
- [ ] 包含1-3个摘要表格（非全面表格）
- [ ] 所有表格都有清晰的列标题
- [ ] 所有表格都有标题行底纹
- [ ] 所有表格底部都有来源行
- [ ] 估计表格显示旧与新，带变化列

**图表：**
- [ ] 文档中嵌入8-12个图表
- [ ] 所有图表上方有"图X - [标题]"说明
- [ ] 所有图表下方有"来源：[来源]"行
- [ ] 图表聚焦于季度趋势
- [ ] 图表突出变化（超出/低于预期，修订）
- [ ] 图表使用专业样式

### 引用与来源检查清单 ⭐⭐⭐ 必须

**图表与表格引用：**
- [ ] 每个图表都有带文档名称和日期的具体来源
- [ ] 每个表格都有带文档引用的具体来源
- [ ] 来源引用在适用时包含页码或幻灯片编号

**超出/低于预期引用：**
- [ ] 超出/低于预期分析引用共识来源（Wind、TuShare、同花顺、东方财富等）
- [ ] 共识来源包含"截至"日期（盈利前收盘）
- [ ] 公司报告的业绩引用至业绩预告/快报或年报/半年报/季报

**指引引用：**
- [ ] 当前指引引用至盈利电话会议记录或发布
- [ ] 先前指引引用至上一季度的材料
- [ ] 当前和先前指引来源均超链接

**统计数据与指标：**
- [ ] 关键统计数据有带来源的脚注
- [ ] 脚注引用具体文档和页码/幻灯片编号
- [ ] 管理层引述注明发言人姓名和来源文档

**超链接：** ⭐⭐⭐ 关键
- [ ] 所有URL都是可点击超链接（非纯文本）
- [ ] 超链接使用有意义的显示文本
- [ ] Word文档中使用蓝色、带下划线的超链接格式
- [ ] 超链接经过测试且可用（Ctrl+点击打开正确页面）
- [ ] 所有国内财报链接至证监会指定信息披露媒体（巨潮/披露易）
- [ ] 所有盈利材料都超链接（发布、记录、演示）
- [ ] 上一季度材料超链接以供比较
- [ ] 文档中任何地方都不显示原始URL

**来源部分：**
- [ ] 报告末尾包含"来源与参考"部分
- [ ] 部分列出所有带日期的盈利材料
- [ ] 所有材料都有可点击超链接
- [ ] 列出共识数据源（即使订阅数据无链接）
- [ ] 包含前期参考

### 准确性检查清单

**数字准确性：**
- [ ] 数字与公司报告的业绩完全匹配
- [ ] 所有计算中的数学正确
- [ ] 估计变化计算正确
- [ ] 估值数学准确
- [ ] 图表与文本描述匹配

**事实准确性：**
- [ ] 股票代码无拼写错误
- [ ] 公司名称无拼写错误
- [ ] 日期当前且准确
- [ ] 季度/年份引用正确
- [ ] 年份标记正确（A表示实际，E表示估计）

### 及时性检查清单

**发布时间：**
- [ ] 报告在盈利发布后24-48小时内发布
- [ ] 如果超过48小时，注明为"延迟反应"
- [ ] ✅ **通过搜索最近盈利验证所有数据来自最新季度**
- [ ] ✅ **未依赖知识截止日期 - 积极搜索当前数据**
- [ ] 共识估计为盈利前（非盈利后）
- [ ] 不包含过时信息
- [ ] 盈利发布日期在过去1-3个月内（不超过6个月）

### 写作风格检查清单

**清晰与直接：**
- [ ] 以数字开头（"收入增长15%至12亿美元"而非"强劲收入"）
- [ ] 使用"vs."而非"versus"
- [ ] 全程直接简洁
- [ ] 聚焦于新内容（不重复公司基础知识）
- [ ] 避免模糊语言（"强劲表现"而不量化）

**专业标准：**
- [ ] 保持机构语调
- [ ] 全程术语一致
- [ ] 无非正式语言
- [ ] 正确的财务符号

## 交付前最终检查

在向用户发送报告前，快速检查以下内容：

### 5分钟最终审查：
1. **第1页**：评级清晰？目标价已更新？关键要点有说服力？
2. **数字**：报告的业绩与公司新闻稿完全匹配？
3. **引用**：抽查3-4个图表/表格 - 都有带可点击超链接的来源？
4. **估计**：清晰显示旧与新？变化已解释？
5. **图表**：所有8-12个都已嵌入？都已编号并加说明？
6. **长度**：是8-12页（不是6页，不是15页）？
7. **超链接**：测试3-4个超链接 - Ctrl+点击是否有效？
8. **及时性**：是否在盈利后48小时内发布？

如果所有项目都通过检查，报告即可交付。

## 摘要交付格式

向用户交付完成的报告时，提供以下摘要：

```
[公司] 第[X]季度 [年份] 盈利更新完成

业绩：[超出预期 / 符合预期 / 低于预期]
- 收入：X.X亿美元（超出/低于XX百万美元或X%）
- 每股收益：X.XX美元（超出/低于X.XX美元）

关键要点：
■ [要点1]
■ [要点2]
■ [要点3]

更新后的估计：
- [年份]财年估计收入：XX.X亿美元（之前：XX.X亿美元，[+/-]X%）
- [年份]财年估计每股收益：X.XX美元（之前：X.XX美元，[+/-]X%）

评级：[维持 / 上调 / 下调] [评级]
目标价：XXX美元（之前：XXX美元） - [+/-]XX%上涨空间

交付物：
✓ 8-12页盈利更新报告（DOCX）
✓ 8-12个嵌入图表
✓ 带新旧比较的更新估计
✓ 带可点击超链接的完整来源部分
✓ [可选：更新的XLS财务模型]

文件：[公司]_Q[X]_[年份]_Earnings_Update.docx
```
