# 早会纪要

## 工作流程

### 步骤1：隔夜动态

扫描覆盖范围内的相关事件：

**业绩与指引**
- 有覆盖公司隔夜或盘前发布业绩吗？
- 业绩惊喜（收入、EPS、关键指标的超预期/不及预期）
- 指引变化（上调、下调、维持）

**新闻与事件**
- 并购公告或传闻
- 管理层变动
- 产品发布或监管决定
- 竞争对手分析师评级上调/下调
- 影响行业的宏观数据或政策变化

**市场背景**
- 隔夜期货/盘前走势
- 行业ETF表现
- 相关大宗商品或汇率变动
- 今日关键经济数据发布

### 步骤2：早会纪要格式

保持紧凑——早会纪要应可在2分钟内读完：

---

**[日期] 早会纪要 — [分析师姓名]**
**[行业覆盖]**

**首要推荐：[标题——基金经理需要知道的一件事]**
- 2-3句话说明关键动态及其重要性
- 股票影响：目标价、评级重申/变化

**隔夜/盘前动态**
- [公司A]：业绩/新闻一句话总结 + 我们的观点
- [公司B]：一句话总结 + 我们的观点
- [行业/宏观]：相关行业整体动态

**今日关键事件**
- [时间]：[公司]业绩电话会议
- [时间]：经济数据发布（预期vs我们的观点）
- [时间]：会议或投资者日

**交易想法**（如有）
- [做多/做空] [公司]：1-2句话论点 + 催化剂
- 风险：什么会让这个判断出错

---

### 步骤3：业绩快速点评

如有覆盖公司发布业绩，提供快速反应：

| 指标 | 一致预期 | 实际值 | 超预期/不及预期 |
|------|---------|--------|---------------|
| 收入 | | | |
| EPS | | | |
| [关键指标] | | | |
| 指引 | | | |

**我们的观点**：2-3句话——这对股票是利好还是利空？会改变我们的论点吗？

**行动**：维持/上调/下调评级？调整目标价？

### 步骤4：输出

- 用于邮件/Slack分发的markdown文本
- 如需正式分发则提供Word文档
- 控制在1页以内——基金经理和交易员不会读更多

## 重要说明

- 要有观点——只总结新闻没有观点的早会纪要毫无价值
- 把最重要的放在前面——不要埋没标题
- "无新闻"也是有效的早会纪要——说"隔夜无重大事项，维持现有持仓"
- 区分可操作事件（业绩、并购）和噪音（次要分析师报告、非事件）
- 为你的观点标注时间戳——如果你在早上6点撰写，注明盘前可能在开盘时变化
- 如果你错了，在下一次早会纪要中承认——可信度比每次都对更重要
