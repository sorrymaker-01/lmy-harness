# 投资想法生成

## 工作流程

### 步骤1：定义搜索标准

向用户询问参数：
- **方向**：多头想法、空头想法或两者都要
- **市值**：大型、中型、小型、微型
- **行业**：特定行业或跨行业
- **风格**：价值、成长、质量、特殊情况、事件驱动
- **地理**：美国、国际、全球
- **主题**：任何特定主题角度（AI、回流、人口老龄化等）

### 步骤2：量化筛选

根据风格运行筛选：

**价值筛选**
- 市盈率低于行业中位数
- EV/EBITDA低于历史平均水平
- 自由现金流收益率 >5%
- 市净率低于1.5倍
- 过去90天内内部人买入
- 股息收益率高于市场平均水平

**成长筛选**
- 收入同比增长 >15%
- 盈利同比增长 >20%
- 收入加速（增长率上升）
- 利润率扩大
- 高投资资本回报率 (>15%)
- 强劲的净留存率（SaaS >110%）

**质量筛选**
- 一致的收入增长（5年以上）
- 稳定或扩大的利润率
- 净资产收益率 >15%
- 低负债/权益
- 高自由现金流转换率
- 内部人持股 >5%

**空头筛选**
- 收入下降或增长减速
- 利润率压缩
- 应收账款/库存相对于销售额上升
- 内部人卖出
- 相对于同行的估值溢价无正当理由
- 高空头兴趣且基本面恶化
- 会计红旗（审计师变更、重述）

**特殊情况筛选**
- 最近的IPO/SPAC锁定期到期
- 过去12个月的分拆
- 从重组中恢复的公司
- 激进投资者参与
- 表现不佳公司的管理层变更

### 步骤3：主题扫描

对于主题想法，研究主题并识别受益者：

1. 定义论点（例如，"AI基础设施支出在2026年前加速"）
2. 映射价值链 — 谁直接受益 vs. 间接受益？
3. 识别纯业务 vs. 多元化 exposure
4. 评估哪些名称已经"定价在内" vs. 被低估
5. 寻找市场尚未与主题关联的二阶受益者

### 步骤4：想法展示

对于通过筛选的每个想法，展示：

**[公司名称] — [多头/空头] — [一句话论点]**

| 指标 | 数值 | 与同行比较 |
|------|------|------------|
| 市值 | | |
| EV/EBITDA (NTM) | | |
| P/E (NTM) | | |
| 收入增长 | | |
| EBITDA利润率 | | |
| FCF收益率 | | |

**论点（3-5个要点）：**
- 为什么被错误定价
- 市场遗漏了什么
- 实现价值的催化剂

**关键风险：**
- 什么会使这一想法错误

**建议的后续步骤：**
- 构建完整模型？深入尽职调查？专家电话？

### 步骤5：输出

- 5-10个想法的简短列表，每个想法一页摘要
- 记录筛选标准和方法
- 所有想法的比较表
- 优先列表：哪些想法优先研究

## 重要说明

- 筛选表面候选人，而非结论 — 每个筛选输出都需要基本面工作
- 最好的想法通常来自交叉点（例如，因暂时逆风而以价值价格交易的优质公司）
- 避免拥挤交易 — 检查所有权数据、空头兴趣和有多少分析师覆盖该名称
- 逆向想法需要催化剂 — 提前但没有催化剂与错误相同
- 随时间跟踪想法命中率 — 哪些筛选和方法产生最佳想法？
- 空头想法需要更高的信心 — 时机更难且风险不对称
