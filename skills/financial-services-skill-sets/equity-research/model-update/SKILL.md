# 模型更新

## 工作流程

### 步骤1：识别变化内容

确定更新触发因素：
- **业绩发布**：需要填入新的季度实际数据
- **指引变化**：公司更新了前瞻展望
- **预测修正**：分析师基于新数据改变假设
- **宏观更新**：利率、汇率、大宗商品价格发生变化
- **事件驱动**：并购、重组、新产品、管理层变动

### 步骤2：填入新数据

#### 业绩发布后
使用报告的实际数据更新模型：

| 科目 | 原预测值 | 实际值 | 差异 | 备注 |
|------|---------|--------|------|------|
| 收入 | | | | |
| 毛利率 | | | | |
| 营业费用 | | | | |
| EBITDA | | | | |
| EPS | | | | |
| [关键指标1] | | | | |
| [关键指标2] | | | | |

**分部详情**（如适用）：
- 更新每个分部的收入和利润率
- 注意任何分部组合变化

**资产负债表/现金流量表更新**：
- 现金和债务余额
- 股份数量（回购、稀释）
- 资本支出实际值vs预测值
- 营运资本变化

### 步骤3：修正前瞻预测

基于新数据，调整前瞻预测：

| | 旧财年预测 | 新财年预测 | 变化 | 旧下一财年 | 新下一财年 | 变化 |
|---|-----------|-----------|------|------------|------------|------|
| 收入 | | | | | | |
| EBITDA | | | | | | |
| EPS | | | | | | |

**关键假设变化：**
- 你正在改变哪些假设，为什么？
- 收入增长率：旧值 → 新值（原因）
- 利润率假设：旧值 → 新值（原因）
- 任何新增项目（重组费用、一次性收益等）

### 步骤4：估值影响

使用更新后的预测重新计算估值：

| 估值方法 | 原值 | 更新值 | 变化 |
|---------|------|--------|------|
| DCF公允价值 | | | |
| 市盈率（NTM EPS × 目标倍数） | | | |
| EV/EBITDA（NTM EBITDA × 目标倍数） | | | |
| **目标价** | | | |

### 步骤5：总结与行动

**预测变化总结：**
- 一段话：什么变化了、为什么、对股票意味着什么
- 这是改变论点的事件还是噪音？

**评级/目标价：**
- 维持或改变评级？
- 新目标价（如有变化）及方法论
- 相对当前价格的上涨/下跌空间

### 步骤6：输出

- 更新后的Excel模型（如用户提供现有模型）
- 预测变化总结（markdown或Word格式）
- 更新后的目标价推导

## 重要说明

- 始终将你的预测与公司报告数据核对后再进行前瞻预测
- 注意任何非经常性项目，以及你的预测是GAAP还是调整后数据
- 追踪你的预测修正历史——它展示了你的分析进展
- 如果季度数据波动较大，在预测变化中区分信号与噪音
- 更新后检查一致预期——你修正后的预测与市场相比如何？
- 股份数量很重要——股权激励、可转债或回购带来的稀释会显著影响EPS
