# 盈利预览

## 工作流程

### 步骤1：收集背景信息

- 确定公司和报告季度
- 通过网络搜索获取共识估计（收入、每股收益、关键细分指标）
- 查找盈利发布日期和时间（盘前 vs. 盘后）
- 回顾公司上一季度盈利电话会议的任何指引或评论

### 步骤2：关键指标框架

为公司构建特定的"需要关注"框架：

**财务指标：**
- 收入 vs. 共识（总体和按细分）
- 每股收益 vs. 共识
- 利润率（毛利润、营业利润、净利润）— 是扩大还是收缩？
- 自由现金流
- 前瞻性指引 vs. 共识

**运营指标**（行业特定）：
- 科技/SaaS：ARR、净留存率、RPO、客户数量
- 零售：同店销售、客流量、购物篮大小
- 工业：未完成订单、订单出货比、价格 vs. 销量
- 金融：净息差、信贷质量、贷款增长、费用收入
- 医疗保健：处方量、患者数量、管线更新

### 步骤3：情景分析

构建3种情景及其对股价的影响：

| 情景 | 收入 | 每股收益 | 关键驱动因素 | 股价反应 |
|------|------|---------|--------------|----------|
| 看涨 | | | | |
| 基准 | | | | |
| 看跌 | | | | |

对于每种情景：
- 运营上需要发生什么
- 管理层的哪些评论会暗示这种情景
- 历史背景 — 股票在类似业绩公布时如何波动？

### 步骤4：催化剂检查清单

识别将决定股票反应的3-5个因素：

1. [指标] vs. [共识/耳语数字] — 为什么重要
2. [指引项目] — 买方期望听到什么
3. [叙事转变] — 任何战略变化、并购、重组

### 步骤5：输出

单页盈利预览，包含：
- 公司、季度、盈利日期
- 共识估计表格
- 需要关注的关键指标（按重要性排序）
- 看涨/基准/看跌情景表格
- 催化剂检查清单
- 交易设置：近期股票表现、期权隐含波动

## 重要说明

- 共识估计会变化 — 始终注明估计的来源和日期
- 来自买方调查的"耳语数字"通常比公布的共识更相关
- 历史盈利反应有助于校准预期（搜索"[公司]盈利反应历史"）
- 期权隐含波动告诉您市场的预期 — 将其与您的情景进行比较
