# 国内财务报告数据提取参考

**使用说明：** 仅当模型模板特别要求从国内财务报告（年报、半年报、季报）中提取数据时，才参考此文件。对于直接提供数据或使用其他数据源的模板，不需要此参考。

---

## 从国内财务报告提取数据

使用国内上市公司数据填充模型模板时，直接从国内财务报告中提取财务数据。

### 步骤1：查找财务报告

1. **A股上市公司**：
   - 巨潮资讯网：http://www.cninfo.com.cn/new/disclosure/stock
   - 搜索股票代码或公司名称
   - 下载最新年报/半年报/季报

2. **港股上市公司**：
   - 港交所披露易：https://www.hkexnews.hk
   - 搜索股票代码或公司名称
   - 下载最新年报/半年报/季报

### 步骤2：识别报告货币

提取数据前，确认报告货币：
- 封面页或标题查看报告货币
- 查看报表标题（如"单位：人民币元"）
- 查看附注1（会计政策及会计估计变更）

**常见货币标识**

| 标识 | 货币 |
|------|------|
| 人民币元、人民币 | CNY |
| 港元、港币 | HKD |
| 美元 | USD |
| 欧元 | EUR |
| 英镑 | GBP |

将模型货币设置为与报告一致，并在假设工作表中记录。

### 步骤3：查找财务报表

在国内财务报告中查找：
- **合并资产负债表**
- **合并利润表**
- **合并现金流量表**
- **财务报表附注**（用于明细数据）

### 步骤4：数据提取映射

**利润表（从合并利润表提取）**
- 营业收入
- 营业成本
- 毛利润
- 销售费用
- 管理费用
- 研发费用
- 财务费用
- 营业利润
- 利润总额
- 所得税费用
- 净利润

**资产负债表（从合并资产负债表提取）**
- 流动资产
  - 货币资金
  - 应收账款
  - 存货
- 非流动资产
  - 固定资产
  - 无形资产
  - 在建工程
- 流动负债
  - 短期借款
  - 应付账款
- 非流动负债
  - 长期借款
  - 应付债券
- 所有者权益
  - 实收资本
  - 资本公积
  - 未分配利润

**现金流量表（从合并现金流量表提取）**
- 经营活动产生的现金流量净额
- 投资活动产生的现金流量净额
- 汇率变动对现金的影响
- 现金及现金等价物净增加额

### 步骤5：验证数据

- 核对报表间勾稽关系
- 资产负债表的期末数 = 利润表的期初数（上期期末）
- 现金流量表净利润 = 经营活动现金流量净额（调整非现金项目后）
- 与附注中的数据交叉核对
