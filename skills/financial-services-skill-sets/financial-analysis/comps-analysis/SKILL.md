# 可比公司分析

## ⚠️ 重要：数据源优先级（请先阅读）

**请始终遵循以下数据源层次结构：**

1. **首先：检查MCP数据源** - 如果Wind MCP、TuShare MCP、Choice MCP或Daloopa MCP可用，请仅使用它们获取财务和交易信息
2. **如果上述MCP数据源可用，请勿使用网络搜索**
3. **仅当MCP不可用时：** 才使用Wind终端、TuShare、同花顺、国内财务报告或其他机构数据源
4. **永远不要将网络搜索作为主要数据源** - 它缺乏机构级分析所需的准确性、审计跟踪和可靠性

**重要性：** MCP来源提供经过验证的机构级数据，并带有适当的引用。网络搜索结果可能过时、不准确或对财务分析不可靠。

## 概述
本技能教授Claude构建机构级可比公司分析，结合运营指标、估值倍数和统计基准。输出是一个结构化的Excel/电子表格，通过同行比较实现明智的投资决策。

**参考材料与情境化：**

`examples/comps_example.xlsx`中提供了可比公司分析示例。使用此技能目录中的此示例或其他示例文件时，请智能使用：

**示例的正确使用方式：**
- 理解结构层次（各部分如何流转）
- 掌握预期的严谨程度（统计深度、文档标准）
- 学习原则（清晰的标题、透明的公式、审计跟踪）

**示例的错误使用方式：**
- 精确复制格式或指标
- 不考虑上下文地复制布局
- 无论受众如何都应用相同的视觉风格

**请始终先问自己：**
1. **"您有首选格式还是我应该调整模板风格？"**
2. **"受众是谁？"**（投资委员会、董事会演示、快速参考、详细备忘）
3. **"核心问题是什么？"**（估值、增长分析、竞争定位、效率）
4. **"背景是什么？"**（并购评估、投资决策、行业基准、绩效评估）

**根据具体情况调整：**
- **行业背景**：大型科技巨头需要与新兴SaaS初创公司不同的指标
- **行业特定需求**：尽早添加相关指标（例如，科技行业的云ARR、企业客户、开发者生态系统）
- **公司熟悉度**：知名公司可能需要较少的背景信息，更多关注差异分析
- **决策类型**：并购与持续的投资组合监控需要不同的侧重点

**核心原则：** 使用模板原则（清晰的结构、统计严谨性、透明的公式），但根据上下文改变执行方式。目标是机构质量的分析，而非机构外观的模板。

用户提供的示例和明确偏好始终优先于默认设置。

## 核心理念
**"先构建正确的结构，然后让数据讲述故事。"**

从强制对重要事项进行战略思考的标题开始，输入干净的数据，构建透明的公式，让统计数据自动呈现。一份好的可比分析应该让没有构建它的人也能立即读懂。

## 第1节：文档结构与设置

### 标题块（第1-3行）
```
第1行：[分析标题] - 可比公司分析
第2行：[带股票代码的公司列表] • [公司1 (TICK1)] • [公司2 (TICK2)] • [公司3 (TICK3)]
第3行：截至 [期间] | 除每股金额和比率外，所有数字均以 [百万/十亿美元] 计算
```

**重要性：** 立即建立上下文。任何打开此文件的人都知道他们在看什么、何时创建以及如何解释这些数字。

### 视觉约定标准（可选 - 用户偏好和上传的模板始终优先）

**重要：这些只是建议的默认值。请始终优先考虑：**
1. 用户明确的格式偏好
2. 任何上传模板文件的格式
3. 公司/团队风格指南
4. 这些默认值（仅在没有其他指导时使用）

**建议的字体与排版：**
- **字体系列**：Times New Roman（专业、易读、行业标准）
- **字体大小**：数据单元格11pt，标题12pt
- **粗体文本**：部分标题、公司名称、统计标签

**建议的颜色与底纹：**
- **部分标题**（例如，"运营统计与财务指标"）：
  - 深蓝色背景（#17365D或类似藏青色）
  - 白色粗体文本
  - 跨所有列的整行底纹
- **列标题**（例如，"公司"、"收入"、"利润率"）：
  - 浅蓝色/灰色背景（#D9E2F3或类似淡蓝色）
  - 黑色粗体文本
  - 居中对齐
- **数据行**：
  - 公司数据为白色背景
  - 输入和公式为黑色文本
- **统计行**（最大值、第75百分位等）：
  - 浅灰色背景（#F2F2F2）
  - 黑色文本，标签左对齐

**建议的格式约定：**
- **小数精度**：
  - 百分比：1位小数（12.3%）
  - 倍数：1位小数（13.5x）
  - 美元金额：无小数，千位分隔符（69,632）
  - 以百分比表示的利润率：1位小数（68.7%）
- **边框**：无边框（干净、简约外观）
- **对齐**：所有指标居中对齐，以获得干净、统一的外观
- **单元格尺寸**：所有列宽应均匀/一致，所有行高应一致（创建干净、专业的网格）

**注意：** 如果用户提供模板文件或指定不同的格式，请使用该模板。

## 第2节：运营统计与财务指标

### 核心列（从这些开始）
1. **公司** - 格式一致的名称
2. **收入** - 规模指标（根据上下文可以是LTM、季度或年度）
3. **收入增长** - 同比百分比变化
4. **毛利润** - 收入减去销货成本
5. **毛利率** - GP/收入（基本盈利能力）
6. **EBITDA** - 息税折旧摊销前利润
7. **EBITDA利润率** - EBITDA/收入（运营效率）

### 可选添加（根据行业/目的选择）
- **季度与LTM** - 如果季节性重要，同时包含两者
- **自由现金流** - 对于资本密集型或SaaS业务
- **FCF利润率** - FCF/收入（现金生成效率）
- **净收入** - 对于成熟、盈利的公司
- **营业利润** - 对于D&A变化较大的业务
- **资本支出指标** - 对于资产密集型行业
- **40法则** - 专门适用于SaaS（增长率% + 利润率%）
- **FCF转换率** - 用于盈利质量分析（高级）

### 公式示例（以第7行为例）
```excel
// 核心比率 - 这些始终计算
毛利率 (F7): =E7/C7
EBITDA利润率 (H7): =G7/C7

// 可选比率 - 相关时包含
FCF利润率: =[FCF]/[收入]
净利润率: =[净收入]/[收入]
40法则: =[增长率%]+[FCF利润率%]
```

**黄金法则：** 每个比率应该是[某指标]/[收入]或[某指标]/[本表中的某指标]。保持简单。

### 统计块（公司数据之后）

**重要：为所有可比指标（比率、利润率、增长率、倍数）添加统计公式。**

```
[留一个空白行用于视觉分隔]
- 最大值: =MAX(B7:B9)
- 第75百分位: =QUARTILE(B7:B9,3)
- 中位数: =MEDIAN(B7:B9)
- 第25百分位: =QUARTILE(B7:B9,1)
- 最小值: =MIN(B7:B9)
```

**需要统计的列（可比指标）：**
- 收入增长率%、毛利率%、EBITDA利润率%、每股收益
- EV/收入、EV/EBITDA、市盈率、股息收益率%、贝塔系数

**不需要统计的列（规模指标）：**
- 收入、EBITDA、净收入（绝对规模因公司规模而异）
- 市值、企业价值（不同规模公司之间不可比）

**注意：** 在公司数据和统计行之间添加一个空白行用于视觉分隔。不要添加"行业统计"或"估值统计"标题行。

**百分位的重要性：** 它们显示分布，而不仅仅是平均值。第75百分位倍数告诉您"优质"公司的交易水平。

## 第3节：估值倍数与投资指标

### 核心估值列（从这些开始）
1. **公司** - 与运营部分相同的顺序
2. **市值** - 当前市场估值
3. **企业价值** - 市值 ± 净债务/现金
4. **EV/收入** - 市场为每美元销售额支付的价格
5. **EV/EBITDA** - 市场为每美元收益支付的价格
6. **市盈率** - 价格相对于净收益

### 可选估值指标（根据上下文选择）
- **FCF收益率** - FCF/市值（用于现金导向分析）
- **PEG比率** - 市盈率/增长率（用于成长型公司）
- **市净率** - 市场价值与账面价值（用于资产密集型业务）
- **ROE/ROA** - 回报指标（用于盈利能力比较）
- **收入/EBITDA CAGR** - 历史增长率（用于趋势分析）
- **资产周转率** - 收入/资产（用于运营效率）
- **负债/权益** - 杠杆（用于资本结构分析）

**关键原则：** 包含对您所在行业重要的3-5个核心倍数。不要仅仅因为可以就包含所有可能的指标。

### 公式示例
```excel
// 核心倍数 - 始终包含这些
EV/收入: =[企业价值]/[LTM收入]
EV/EBITDA: =[企业价值]/[LTM EBITDA]
市盈率: =[市值]/[净收入]

// 可选倍数 - 数据可用时包含
FCF收益率: =[LTM FCF]/[市值]
PEG比率: =[市盈率]/[增长率%]
```

### 交叉引用规则
**重要：** 估值倍数必须引用运营指标部分。永远不要重复输入相同的原始数据。如果收入在C7中，那么EV/收入公式应引用C7。

### 统计块
与运营部分相同的结构：每个指标的最大值、第75百分位、中位数、第25百分位、最小值。在公司数据和统计之间添加一个空白行用于视觉分隔。不要添加"估值统计"标题行。

## 第4节：注释与方法论文档

### 必要组件

**数据源与质量：**
-- 数据来自哪里？（Wind MCP、TuShare MCP、Choice MCP、Daloopa MCP、同花顺、国内财务报告）
- 如何验证？（与年报/半年报交叉核对）
- 注意：如果可用，请优先使用MCP数据源（Wind、TuShare、Choice、Daloopa）以获得更好的准确性和可追溯性

**关键定义：**
- EBITDA计算方法（毛利润 + D&A，或营业利润 + D&A）
- 自由现金流公式（经营现金流 - 资本支出）
- 特殊指标解释（40法则、FCF转换率）
- 时间段定义（LTM、CAGR计算期间）

**估值方法论：**
- 企业价值如何计算？（市值 + 净债务）
- 使用了什么增长率？（历史CAGR、前瞻性估计）
- 是否进行了任何调整？（排除一次性项目，标准化利润率）

**分析框架：**
- 投资论点是什么？（云/SaaS效率）
- 哪些指标最重要？（现金生成、资本效率）
- 读者应如何解释统计数据？（百分位提供上下文）

## 第5节：选择正确的指标（决策框架）

### 从"我要回答什么问题？"开始

**"哪家公司被低估？"**
→ 关注：EV/收入、EV/EBITDA、市盈率、市值
→ 跳过：运营细节、增长指标

**"哪家公司效率最高？"**
→ 关注：毛利率、EBITDA利润率、FCF利润率、资产周转率
→ 跳过：规模指标、绝对美元金额

**"哪家公司增长最快？"**
→ 关注：收入增长率%、EBITDA CAGR、用户/客户增长
→ 跳过：利润率指标、杠杆比率

**"哪家是最佳现金生成者？"**
→ 关注：FCF、FCF利润率、FCF转换率、资本支出强度
→ 跳过：EBITDA、市盈率

### 行业特定指标选择

**软件/SaaS：**
必备：收入增长、毛利率、40法则
可选：ARR、净美元留存率、CAC回收期
跳过：资产周转率、库存指标

**制造业/工业：**
必备：EBITDA利润率、资产周转率、资本支出/收入
可选：ROA、库存周转率、未完成订单
跳过：40法则、SaaS指标

**金融服务：**
必备：ROE、ROA、效率比率、市盈率
可选：净息差、贷款损失准备金
跳过：毛利率、EBITDA（对银行无意义）

**零售/电子商务：**
必备：收入增长、毛利率、库存周转率
可选：同店销售、客户获取成本
跳过：重研发或资本支出指标

### "5-10法则"

**5个运营指标** - 收入、增长、2-3个利润率/效率指标
**5个估值指标** - 市值、企业价值、3个倍数
**= 共10列** - 足以讲述故事，不会过多导致失去主线

如果您有超过15个指标，可能包含了噪音。请严格编辑。

## 第6节：最佳实践与质量检查

### 开始之前
1. **定义同行组** - 公司必须真正可比（相似的商业模式、规模、地理位置）
2. **选择正确的期间** - LTM平滑季节性；季度显示趋势
3. **预先标准化单位** - 百万与十亿的决定影响一切
4. **映射数据源** - 知道每个数字的来源

### 构建过程中
1. **首先输入所有原始数据** - 在编写公式之前完成蓝色文本
2. **为所有硬编码输入添加单元格注释** - 右键单击单元格 → 插入注释 → 记录来源或假设

   **对于来源数据，准确引用其来源：**
   - 示例："Wind/TuShare终端 - 600519贵州茅台，访问时间2024-10-02"
   - 示例："2024年年度报告，第42页，项目'营业收入'"
   - 示例："截至2024-10-02的Wind/TuShare一致预期"
   - **尽可能包含超链接**：右键单击单元格 → 链接 → 粘贴财务报告、数据源或报告的URL

   **对于假设，解释推理：**
   - 示例："基于同行中位数假设15%的EBITDA利润率，公司未披露"
   - 示例："估计企业价值为市值 + 5000万美元净债务（来自第三季度资产负债表，第四季度尚未可用）"
   - 示例："前瞻性市盈率基于华尔街共识每股收益3.45美元（12位分析师估计的平均值）"

   **重要性：** 实现审计跟踪、数据验证、假设透明度和未来更新
3. **逐行构建公式** - 在继续之前测试每个计算
4. **对标题使用绝对引用** - $C$6锁定标题行
5. **一致格式化** - 百分比为百分比，而非小数
6. **添加条件格式** - 自动突出显示异常值

### 合理性检查
- **利润率测试**：毛利率 > EBITDA利润率 > 净利润率（根据定义始终为真）
- **倍数合理性**：
  - EV/收入：通常为0.5-20倍（因行业而异）
  - EV/EBITDA：通常为8-25倍（跨行业相当一致）
  - 市盈率：通常为10-50倍（取决于增长率）
- **增长-倍数相关性**：较高的增长通常意味着较高的倍数
- **规模-效率权衡**：较大的公司通常具有更好的利润率（规模优势）

### 需避免的常见错误
❌ 在公式中混合使用市值和企业价值
❌ 分子和分母使用不同的时间段（LTM与季度）
❌ 在公式中硬编码数字而不是单元格引用
❌ **硬编码输入没有单元格注释引用来源或解释假设**
❌ 可用时缺少指向财务报告或数据源的超链接
❌ 包含过多无明确目的的指标
❌ 包含不可比公司（不同的商业模式）
❌ 使用未披露的过时数据
❌ 错误计算百分比的平均值（应使用中位数）

## 第6节：高级功能

### 动态标题
对于显示计算的列，使用清晰的单位标签：
```
收入增长（同比）% | EBITDA利润率 | FCF利润率 | 40法则
```

### 百分位分析的好处
除了均值/中位数外，百分位还显示：
- **第75百分位** = "优质"公司在此交易
- **中位数** = 典型市场估值
- **第25百分位** = "折扣"区域

这有助于回答："我们的目标公司与同行相比是高估还是低估？"

### 行业特定修改

**软件/SaaS：**
- 添加：ARR、净美元留存率、CAC回收期
- 强调：40法则、FCF利润率、毛利率>70%

**医疗保健：**
- 添加：研发/收入、管线价值、监管状态
- 强调：EBITDA利润率、增长率、报销风险

**工业：**
- 添加：未完成订单、订单簿趋势、地理分布
- 强调：ROIC、资产周转率、周期性调整

**消费品：**
- 添加：同店销售、客户获取成本、品牌价值
- 强调：收入增长、毛利率、库存周转率

## 第7节：工作流程与实用技巧

### 分步流程
1. **设置结构**（30分钟）
   - 创建所有标题
   - 格式化单元格（蓝色用于输入，黑色用于公式）
   - 锁定单位和日期引用

2. **收集数据**（60-90分钟）
   - 从主要来源获取（如果可用，Wind MCP、TuShare MCP、Choice MCP、Daloopa MCP；否则同花顺、国内财务报告）
   - 以蓝色输入所有原始数字
   - 在注释部分记录来源

3. **构建公式**（30分钟）
   - 从简单比率开始（利润率）
   - 进展到倍数（EV/收入）
   - 添加交叉检查（利润率是否合理？）

4. **添加统计数据**（15分钟）
   - 为所有列复制公式结构
   - 验证范围正确（B7:B9，而非B7:B10）
   - 检查百分位逻辑

5. **质量控制**（30分钟）
   - 运行合理性检查
   - 验证公式引用
   - 检查#DIV/0!或#REF!错误
   - 与已知基准进行比较

6. **文档**（15分钟）
   - 完成注释部分
   - 添加数据源
   - 定义方法论
   - 为分析添加日期戳

### 专业技巧
- **保存模板**：构建一次，永久重用
- **为异常值着色**：值>2个标准差的条件格式
- **链接到源文件**：超链接到Wind截图或财务报告
- **版本控制**：保存为"Comps_v1_2024-12-15"，带有清晰的日期
- **协作审查**：让其他人检查您的公式

## 第10节：红旗与警告信号

### 数据质量问题
🚩 不一致的时间段（混合季度和年度）  
🚩 无解释的缺失数据  
🚩 数据源之间的显著差异（>10%方差）

### 估值红旗
🚩 负EBITDA公司使用EBITDA倍数估值（应使用收入倍数）  
🚩 无高增长故事的市盈率>100倍  
🚩 不符合行业逻辑的利润率

### 可比性问题
🚩 不同的财年结束（导致时间问题）  
🚩 混合纯业务和企业集团  
🚩 被标记为"可比"的显著不同商业模式

**如有疑问，请排除该公司。** 拥有3个完美的可比公司比拥有6个有问题的可比公司更好。

## 第11节：公式参考指南

### 必要的Excel公式
```excel
// 统计函数
=AVERAGE(range)          // 简单均值
=MEDIAN(range)           // 中间值
=QUARTILE(range, 1)      // 第25百分位
=QUARTILE(range, 3)      // 第75百分位
=MAX(range)              // 最大值
=MIN(range)              // 最小值
=STDEV.P(range)          // 标准差

// 财务计算
=B7/C7                   // 简单比率（利润率）
=SUM(B7:B9)/3            // 多家公司的平均值
=IF(B7>0, C7/B7, "N/A")  // 条件计算
=IFERROR(C7/D7, 0)       // 处理除零错误

// 跨表引用
='Sheet1'!B7             // 引用另一个工作表
=VLOOKUP(A7, Table1, 2)  // 从数据表查找
=INDEX(MATCH())          // 高级查找

// 格式化
=TEXT(B7, "0.0%")        // 格式化为百分比
=TEXT(C7, "#,##0")       // 千位分隔符
```

### 常见比率公式
```excel
毛利率 = 毛利润 / 收入
EBITDA利润率 = EBITDA / 收入
FCF利润率 = 自由现金流 / 收入
FCF转换率 = 自由现金流 / 经营现金流
ROE = 净收入 / 股东权益
ROA = 净收入 / 总资产
资产周转率 = 收入 / 总资产
负债/权益 = 总负债 / 股东权益
```

## 关键原则总结

1. **结构驱动洞察** - 正确的标题强制正确的思考
2. **少即是多** - 5-10个重要指标胜过20个不重要的指标
3. **为您的问题选择指标** - 估值分析 ≠ 效率分析
4. **统计数据显示模式** - 中位数/百分位比平均值揭示更多
5. **透明度胜于复杂性** - 每个人都理解的简单公式
6. **可比性为王** - 排除比强制使用不良可比公司更好
7. **记录您的选择** - 在注释部分解释使用哪些指标以及原因
