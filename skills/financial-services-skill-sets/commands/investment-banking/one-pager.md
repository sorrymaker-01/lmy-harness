# 单页公司简介命令

为路演材料和交易材料创建专业的单页公司简介。

## 工作流程

### 步骤1：收集公司信息

如果提供了公司名称或股票代码，则使用该信息。否则询问：
- "您想为哪家公司创建简介？"

### 步骤2：检查可用的PPT模板技能

**首先，检查skills目录中是否有现有的ppt-template技能**：

```bash
ls skills/ | grep -E "ppt-template|brand-guidelines"
```

如果存在模板技能（例如`techcorp-ppt-template`、`gs-brand-guidelines`）：
1. 向用户列出可用模板
2. 询问使用哪个模板，或是否需要简洁的专业格式
3. 使用`skill: "[template-name]"`加载选定的模板技能

如果不存在模板技能，询问：
- "您是否有品牌PowerPoint模板文件可以使用？如果有，请提供路径。否则我将使用简洁的专业格式。"

如果提供了模板文件：
1. 分析模板结构以了解布局
2. 使用适当的布局创建单页内容

### 步骤3：加载公司简介技能

使用`skill: "strip-profile"`执行简介创建：

1. **明确需求**：
   - 确认单张幻灯片格式（单页）
   - 询问是否有特定关注领域

2. **研究公司数据**：
   - 公司概况（总部、成立时间、员工、管理层）
   - 业务描述和定位
   - 关键财务数据（收入、EBITDA、利润率、增长）
   - 估值指标（市值、企业价值、倍数）
   - 近期发展和新闻
   - 主要股东（上市公司）

3. **创建公司简介**：
   - 使用4:3宽高比（10" x 7.5"）
   - 四象限布局：
     - 左上：公司概况（项目符号）
     - 右上：业务与定位（项目符号）
     - 左下：关键财务数据（表格）
     - 右下：股价走势图 + 股东或近期新闻
   - 应用公司品牌颜色
   - 在章节标题上添加强调条

### 步骤4：视觉审查

创建幻灯片后：
1. 转换为图像进行审查
2. 检查文本重叠/截断问题
3. 验证所有数据已填充（无占位符）
4. 向用户展示预览以供批准

### 步骤5：交付输出

提供：
1. **PowerPoint文件**（.pptx）- 单页简介
2. **图像预览** - 用于快速审查
3. **摘要** - 包含的关键数据点

```
┌─────────────────────────────────────────────────────────────────┐
│ 公司名称（股票代码）                                    [标志]   │
├────────────────────────────┬────────────────────────────────────┤
│ 公司概况                   │ 业务与定位                         │
│ • 总部、成立时间、员工     │ • 核心业务描述                     │
│ • CEO、CFO                 │ • 主要产品/服务                    │
│ • 市值、行业               │ • 竞争定位                         │
│ • 关键统计数据             │ • 增长驱动因素                     │
├────────────────────────────┼────────────────────────────────────┤
│ 关键财务数据               │ 股价表现 / 股权结构                │
│ ┌──────────────────────┐   │ [1年股价走势图]                    │
│ │ 指标  │ FY24 │ FY25E│   │                                    │
│ │ 收入  │ $XXB │ $XXB │   │ 主要股东：                         │
│ │ EBITDA│ $XXB │ $XXB │   │ • Vanguard: X.X%                   │
│ │ 利润率│ XX%  │ XX%  │   │ • BlackRock: X.X%                  │
│ │ EV/EBITDA│ XXx │ XXx │   │ • State Street: X.X%               │
│ └──────────────────────┘   │                                    │
└────────────────────────────┴────────────────────────────────────┘
来源：公司文件、FactSet
```

## 质量清单

交付前：
- [ ] 所有4个象限已填充真实数据
- [ ] 无剩余占位符文本
- [ ] 已应用公司品牌颜色
- [ ] 所有章节标题有强调条
- [ ] 财务表格格式正确
- [ ] 底部已引用来源
- [ ] 无文本溢出或截断
- [ ] 投资银行质量（GS/MS/JPM标准）
