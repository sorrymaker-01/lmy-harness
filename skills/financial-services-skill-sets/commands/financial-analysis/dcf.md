# DCF估值命令

构建机构级质量的DCF模型，使用可比公司分析来确定估值范围。

## 工作流程

### 步骤1：收集公司信息

如果提供了公司名称或股票代码，则使用该信息。否则询问：
- "您想对哪家公司进行估值？"

### 步骤2：运行可比公司分析

**首先，加载comps-analysis技能**构建交易倍数：

使用`skill: "comps-analysis"`：
1. 识别4-6家可比上市公司
2. 获取运营指标（收入、EBITDA、利润率、增长率）
3. 获取估值倍数（EV/收入、EV/EBITDA、P/E）
4. 计算统计摘要（中位数、25/75分位数）

**从可比公司分析中获取的关键输出：**
- 中位数EV/EBITDA倍数 → 用于确定终值退出倍数
- 中位数EV/收入倍数 → DCF输出的合理性检查
- 同行增长率 → 收入预测的基准
- 同行利润率 → 利润率假设的基准

### 步骤3：构建DCF模型

**加载dcf-model技能**构建估值：

使用`skill: "dcf-model"`：
1. 收集历史财务数据和市场数据
2. 构建收入预测（熊市/基准/牛市情景）
3. 建模运营费用和自由现金流
4. 使用CAPM计算WACC
5. 折现现金流并计算终值
6. 桥接到股权价值和隐含股价

**使用可比公司信息确定DCF假设：**

| 可比公司输出 | DCF输入 |
|--------------|---------|
| 同行中位数EV/EBITDA | 终值退出倍数范围 |
| 同行25-75分位EV/EBITDA | 敏感性分析范围 |
| 同行中位数增长率 | 收入假设基准 |
| 同行中位数EBITDA利润率 | 终值年份目标利润率 |
| 同行中位数P/E | 交叉验证DCF隐含P/E |

### 步骤4：交叉验证估值

DCF完成后，验证：
1. **DCF隐含EV/EBITDA** vs 同行中位数
   - 如果DCF隐含25倍但同行交易在12倍，调查原因
2. **DCF隐含P/E** vs 同行中位数
3. **终值占EV比例**（应为50-70%）
4. **估值中隐含的增长率** vs 同行增长率

### 步骤5：交付输出

提供：
1. **可比公司分析电子表格**（.xlsx），包含同行交易倍数
2. **DCF模型**（.xlsx），包含：
   - 熊市/基准/牛市情景
   - 敏感性分析表（WACC vs 永续增长率等）
   - 估值摘要，包含隐含上涨/下跌空间
3. **摘要**，说明：
   - 关键估值驱动因素
   - 可比公司如何影响分析
   - 需关注的风险和敏感性

## 示例输出摘要

```
估值摘要：[公司]（[股票代码]）

可比公司分析：
- 可比公司群体：[4-6家可比公司列表]
- 中位数EV/EBITDA：12.5倍（范围：10.2倍 - 15.8倍）
- 中位数EV/收入：3.2倍（范围：2.1倍 - 4.5倍）

DCF估值（基准情景）：
- 隐含股价：$XX.XX
- 当前股价：$YY.YY
- 隐含上涨空间：+XX%

估值交叉验证：
- DCF隐含EV/EBITDA：13.2倍（vs 同行中位数12.5倍）
- DCF隐含P/E：22.4倍（vs 同行中位数20.1倍）
- 终值：占EV的62%（在正常范围内）

关键假设：
- 收入CAGR：X%（vs 同行中位数X%）
- 终值EBITDA利润率：X%（vs 同行中位数X%）
- WACC：X.X%
- 永续增长率：X.X%
```
