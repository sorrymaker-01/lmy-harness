# 可比公司分析命令

构建机构级的可比公司分析，包含运营指标、估值倍数和统计基准。

## 工作流程

### 步骤1：收集公司信息

如果提供了公司名称或股票代码，则使用该信息。否则询问：
- "您想分析哪家公司？"

### 步骤2：加载可比公司分析技能

使用`skill: "comps-analysis"`构建分析：

1. **明确分析目的**：
   - "核心问题是什么？"（估值、效率、增长比较）
   - "受众是谁？"（投资委员会、董事会、快速参考）
   - "您是否有偏好的格式或模板？"

2. **识别可比公司群体**（4-6家可比公司）：
   - 相似的商业模式
   - 相似的规模/市值范围
   - 同一行业/板块
   - 地理位置可比性

3. **收集数据**（优先使用MCP数据源）：
   - 运营指标：收入、增长率、毛利率、EBITDA、EBITDA利润率
   - 估值：市值、企业价值、EV/收入、EV/EBITDA、P/E
   - 基于行业的额外指标（SaaS的40法则等）

4. **构建分析**：
   - 运营统计部分，包含公司数据 + 统计数据（最大值、75分位、中位数、25分位、最小值）
   - 估值倍数部分，包含相同的统计摘要
   - 注释与方法论文档

### 步骤3：创建Excel输出

生成Excel文件，包含：
- 标题区块（分析标题、公司、日期、单位）
- 运营统计与财务指标部分
- 估值倍数部分
- 每个指标的统计摘要
- 记录来源和方法论的注释部分

### 步骤4：交付输出

提供：
1. **Excel文件**（.xlsx）- 可比公司分析
2. **摘要**，重点说明：
   - 可比公司选择理由
   - 关键洞察（谁以溢价/折价交易）
   - 中位数倍数供参考

## 行业特定指标

| 行业 | 额外指标 |
|------|----------|
| 软件/SaaS | ARR、净金额留存率、40法则 |
| 零售 | 同店销售、库存周转率 |
| 金融 | ROE、ROA、效率比率 |
| 制造业 | 资产周转率、资本支出/收入 |
| 医疗健康 | 研发/收入、管线价值 |

## 质量清单

交付前：
- [ ] 4-6家真正可比的公司
- [ ] 一致的时间段（全部LTM或全部财年）
- [ ] 所有公式引用单元格（无硬编码值）
- [ ] 所有硬编码输入有单元格批注注明来源
- [ ] 统计数据包含最大值、75分位、中位数、25分位、最小值
- [ ] 注释部分记录来源和方法论
- [ ] 蓝色 = 输入，黑色 = 公式
- [ ] 合理性检查通过（利润率合理、倍数合理）
