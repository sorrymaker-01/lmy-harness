# 业绩分析命令

创建专业的股权研究业绩更新报告，分析季度业绩结果。

## 工作流程

### 步骤1：收集信息

解析输入以获取：
- 公司名称或股票代码
- 季度（例如，Q3 2024、Q2 FY25）

如果未提供，询问：
- "您想分析哪家公司的业绩？"
- "哪个季度？（例如，Q3 2024）"

### 步骤2：验证时效性

**关键**：在继续之前，验证您拥有最新数据：
1. 搜索"[公司]最新业绩结果[当前年份]"
2. 验证业绩发布在最近3个月内
3. 确认电话会议日期与发布日期匹配

如果数据过时，通知用户并搜索最新信息。

### 步骤3：加载业绩分析技能

使用`skill: "earnings-analysis"`创建报告：

1. **数据收集**（搜索最新）：
   - 业绩公告（新闻稿）
   - 交所/港交所的年报/半年报/季报
   - 业绩说明会记录
   - 投资者演示/补充材料
   - 一致预期（Wind/TuShare/同花顺/东方财富）

2. **超预期/不及预期分析**：
   - 收入vs一致预期：超预期/不及预期$X或X%
   - EPS vs一致预期：超预期/不及预期$X或X%
   - 关键分部业绩vs预期
   - 解释结果差异的原因

3. **关键指标分析**：
   - 按分部/地区划分的收入明细
   - 利润率趋势（毛利率、营业利润率、净利润率）
   - 指引：上调/维持/下调
   - 更新的前瞻预测

4. **生成图表**（8-12张）：
   - 季度收入进展
   - 季度EPS进展
   - 利润率趋势
   - 按分部划分的收入
   - 超预期/不及预期摘要
   - 预测修正
   - 估值图表

5. **创建报告**（8-12页）：
   - 第1页：摘要，包含评级和目标价
   - 第2-3页：详细业绩分析
   - 第4-5页：关键指标和指引
   - 第6-7页：更新投资论点
   - 第8-10页：估值和预测
   - 来源部分，包含可点击超链接

### 步骤4：交付输出

提供：
1. **DOCX报告** - 8-12页业绩更新
2. **摘要**，重点说明：
   - 关键指标的超预期/不及预期
   - 指引变化
   - 论点影响（正面/负面/中性）

## 报告结构参考

```
第1页：业绩摘要
┌─────────────────────────────────────────────────────────────────┐
│ [公司] Q3 2024业绩更新                                          │
│ 评级：买入 | 目标价：$XXX（原$XXX）                              │
├─────────────────────────────────────────────────────────────────┤
│ 关键要点                                                        │
│ • 收入超预期X%，得益于[分部]强劲表现                            │
│ • EPS超预期$X.XX，受利润率扩张驱动                              │
│ • 财年指引上调至$X.XX-$X.XX（原$X.XX-$X.XX）                    │
│ • 论点不变；维持买入评级                                         │
├─────────────────────────────────────────────────────────────────┤
│ 业绩快照                                                        │
│ ┌─────────────┬──────────┬──────────┬──────────┐               │
│ │ 指标        │ 实际     │ 一致预期 │ 超预期   │               │
│ │ 收入        │ $X.XXB   │ $X.XXB   │ +X.X%    │               │
│ │ EPS         │ $X.XX    │ $X.XX    │ +$X.XX   │               │
│ │ 毛利率      │ XX.X%    │ XX.X%    │ +XXbps   │               │
│ └─────────────┴──────────┴──────────┴──────────┘               │
└─────────────────────────────────────────────────────────────────┘

第2-3页：详细业绩
- 分部分析
- 地区明细
- 超预期/不及预期的关键驱动因素

第4-5页：指标与指引
- 利润率分析
- 全年指引对比
- 更新的季度预测

第6-7页：论点更新
- 变化内容
- 风险和催化剂
- 投资建议

第8-10页：估值
- 如有重大变化，更新DCF/可比公司
- 目标价论证
- 情景分析

来源部分（含可点击超链接）：
- 业绩公告：[超链接]
- 季度报告：[巨潮/披露易超链接]
- 业绩说明会纪要：[超链接]
- 一致预期：Wind/TuShare，截至[日期]
```

## 质量清单

交付前：
- [ ] 业绩数据来自最新季度（非过时）
- [ ] 超预期/不及预期已用具体数字量化
- [ ] 所有图表已嵌入（共8-12张）
- [ ] 来源部分包含可点击超链接
- [ ] 每个图表/表格有来源引用
- [ ] 指引变化已清楚记录
- [ ] 评级和目标价在开头明确说明
- [ ] 8-12页，3,000-5,000字
