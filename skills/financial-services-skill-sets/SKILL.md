# 金融服务技能集合

专业金融服务从业者的 AI 技能集合，适用于 A股、港股及中概股市场。

## 概述

本项目提供完整的金融服务领域 AI 技能，覆盖：
- **股票研究分析** - 盈利分析、首次覆盖、行业研究
- **投资银行并购** - Teaser、CIM、买家名单、估值
- **财务建模分析** - DCF、可比公司、三表模型、LBO
- **私募股权投资** - 尽职调查、回报分析、价值创造
- **财富管理顾问** - 客户报告、资产配置、税务规划

## 使用方式

### 直接调用具体技能
```
使用 skill: "earnings-analysis" 分析贵州茅台的最新季度业绩
```

### 通过命令调用
```
使用 commands/equity-research/earnings.md 分析 [公司] 的 Q3 2024 业绩
```

## 规范遵循

本项目遵循 [Agent Skills](https://agentskills.io/what-are-skills) 设计规范：

```
volc-financial-services-skills/
├── SKILL.md                    # 技能集合索引（本文件）
├── equity-research/            # 技能领域
│   └── [skill-name]/
│       ├── SKILL.md           # 技能定义
│       ├── references/        # 参考文档
│       ├── scripts/           # 可执行脚本
│       └── assets/            # 模板资源
├── financial-analysis/
├── investment-banking/
├── private-equity/
├── wealth-management/
└── partner-built/              # 第三方技能
```

## 贡献指南

添加新技能时：
1. 在对应领域目录下创建 `[skill-name]/`
2. 创建 `SKILL.md` 文件，包含 name、description
3. 添加可选的 `references/`、`scripts/`、`assets/` 目录
4. 更新本索引文件的技能列表
