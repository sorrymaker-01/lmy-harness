# OpenClaw 金融服务 SKILL 集

将 OpenClaw 转变为金融服务专家的 SKILL 集——投资银行、股权研究、私募股权和财富管理。专为金融专业人士打造，提供端到端的金融分析和交易支持能力。

## 为什么使用 OpenClaw 金融服务 SKILL 集

OpenClaw 让您设定目标，AI 交付完成的专业工作。SKILL 集让您走得更远：告诉 OpenClaw 您的公司如何进行分析、从哪些数据源获取数据、如何处理关键工作流程，以及暴露哪些命令——让您的团队获得更好、更一致的成果。

每个 SKILL 都为特定的金融服务工作流程打包了专业知识、最佳实践和分步指南。开箱即用，它们为 OpenClaw 帮助该角色的任何人提供了强有力的起点。真正的力量来自于为您公司定制它们——您的模型、您的模板、您的流程——让 OpenClaw 就像为您的团队构建的一样工作。

## 什么是 OpenClaw 金融服务？

OpenClaw 金融服务是构建在 OpenClaw 平台之上的综合解决方案，具有专门的财务分析能力。它将 OpenClaw 连接到金融专业人士日常使用的数据源和工具——无需在多个浏览器标签之间切换，并改进来源验证以减少手动数据收集的错误风险。

## 端到端工作流程

这些 SKILL 不仅仅是工具的集合——它们支持跨越研究、分析、建模和输出创建的完整工作流程：

- **研究到报告**：从数据提供商获取实时数据，分析业绩结果，生成可发布的股权研究报告——全部在单个会话中完成
- **电子表格分析**：构建可比公司分析、DCF 模型和 LBO 模型，作为具有实时公式、敏感性分析表和行业标准格式的完整功能 Excel 工作簿
- **财务建模**：从国内财务报告填充三表模型，与同行数据交叉核对假设，进行情景压力测试——内置蓝/黑/绿颜色编码约定
- **交易材料**：起草 CIM、Teaser 和流程函，然后使用您公司的品牌 PowerPoint 模板生成演示文稿幻灯片和公司简介
- **投资组合到演示**：筛选机会、运行尽职调查清单、构建 IC 备忘录、跟踪投资组合 KPI——从数据到交付物的无缝流转

每个工作流程都将上游数据源连接到下游输出（Excel、PowerPoint、Word），让您从问题到完成的工作成果无需切换上下文。

## 快速开始

### 在 OpenClaw 中使用

1. 克隆或下载本仓库到本地
2. 在 OpenClaw 中导入 SKILL 集
3. 首先启用 **财务分析** 核心 SKILL
4. 然后根据需要启用其他专业 SKILL

使用示例命令：

```bash
/comps [公司]                # 可比公司分析
/dcf [公司]                  # DCF 估值模型
/earnings [公司] [季度]      # 业绩后更新报告
/one-pager [公司]            # 单页公司简介
/ic-memo [项目名称]          # 投资委员会备忘录
/source [标准]               # 交易寻源
/client-review [客户]        # 客户会议准备
```

## SKILL 工作原理

每个 SKILL 遵循相同的结构：

```
skill-name/
├── SKILL.md                   # SKILL 定义和工作流程
├── references/                # 参考资料和最佳实践
└── scripts/                   # 辅助脚本（如有）
```

- **SKILL.md**：编码了 OpenClaw 交付专业质量财务工作所需的领域专业知识、最佳实践和分步工作流程。OpenClaw 在相关时自动调用它们。
- **命令**：是您触发的显式操作（例如 `/comps`、`/earnings`、`/ic-memo`）。
- **参考资料**：包含行业标准、格式指南和工作流程文档，确保输出符合专业标准。

每个组件都是基于文件的——markdown 和 JSON，无需复杂的代码，无需基础设施，无需构建步骤。

## 使其成为您的

这些 SKILL 是起点。当您根据公司的实际工作方式进行定制时，它们会变得更有用：

- **添加公司背景** — 将您的术语、交易流程和格式标准放入 SKILL 文件，让 OpenClaw 理解您的世界。
- **使用您的模板** — 配置 OpenClaw 使用您公司的品牌 PowerPoint 布局，让每份演示文稿都符合您的风格指南。
- **调整工作流程** — 修改 SKILL 说明以匹配您团队实际的分析方式，而非教科书上的方式。
- **构建新 SKILL** — 遵循上述结构为我们尚未覆盖的工作流程创建 SKILL。

随着您的团队构建和分享 SKILL，OpenClaw 成为跨职能专家。您定义的上下文被融入每个相关交互中，领导者可以花更少时间执行流程，更多时间改进流程。
