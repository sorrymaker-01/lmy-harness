# 回报分析

## 工作流程

### 步骤1：收集交易输入

询问（或从先前分析中提取）：

**进入：**
- 进入EBITDA（过去12个月或未来12个月）
- 进入倍数（企业价值/EBITDA）
- 企业价值
- 交易完成时的净债务
-  equity check 规模
- 交易费用和支出

**融资：**
- 高级债务（EBITDA倍数、利率、摊销）
- 次级债务/夹层融资（如有）
- 进入时的总杠杆（EBITDA倍数）
- 股权出资

**运营假设：**
- 收入增长率（年度）
- EBITDA利润率轨迹
- 资本支出占收入的百分比
- 营运资金变化
- 债务偿还计划

**退出：**
- 持有期（年）
- 退出倍数（企业价值/EBITDA）
- 退出EBITDA（根据增长假设计算）

## 关键公式

- **MOIC** = 退出股权价值 / 投入股权
- **IRR** = 求解r：投入股权 × (1 + r)^n = 退出股权价值（调整中期现金流）
- **回报归因**：
  - 增长：(退出EBITDA - 进入EBITDA) × 退出倍数 / 股权
  - 倍数：(退出倍数 - 进入倍数) × 进入EBITDA / 股权
  - 杠杆：持有期内的债务偿还 / 股权

## 重要注意事项

- 在适用情况下，始终同时显示毛回报和扣除费用/carry后的净回报
- 管理层滚投和共同投资会改变equity check — 如相关请询问
- 股息 recap 或中期分配会显著影响IRR — 如有计划请包含
- 不要忘记交易成本（通常为企业价值的2-4%）— 它们会减少第1天的股权价值
- 税务考虑（资产交易vs股票交易，338(h)(10)选择）会对税后回报产生重大影响
