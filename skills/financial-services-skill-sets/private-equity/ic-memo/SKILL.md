# 投资委员会备忘录
## 工作流程

### 步骤1：收集输入

从用户（或会话中的先前分析）收集：

- 公司概述和业务描述
- 行业/市场背景
- 历史财务数据（3-5年）
- 管理层评估
- 交易条款（价格、结构、融资）
- 尽职调查结果（商业、财务、法律、运营）
- 价值创造计划/100天计划
- 回报分析（基准、上行、下行）

### 步骤2：起草备忘录结构

标准IC备忘录格式：

**I. 执行摘要**（1页）
- 公司描述、交易 rationale、关键条款
- 推荐意见和主要回报
- 前3大风险及缓解措施

**II. 公司概述**（1-2页）
- 业务描述、产品/服务
- 客户基础和市场进入策略
- 竞争定位
- 管理团队

**III. 行业与市场**（1页）
- 市场规模和增长
- 竞争格局
- 长期趋势/有利因素
- 监管环境

**IV. 财务分析**（2-3页）
- 历史表现（收入、EBITDA、利润率、现金流）
- 盈利质量调整
- 营运资金分析
- 资本支出需求

**V. 投资论点**（1页）
- 为什么这是一个有吸引力的投资（3-5个支柱）
- 价值创造杠杆（有机增长、利润率扩张、并购、估值倍数扩张）
- 100天优先事项

**VI. 交易条款与结构**（1页）
- 企业价值和隐含倍数
- 资金来源与用途
- 资本结构/杠杆
- 关键法律条款

**VII. 回报分析**（1页）
- 基准、上行和下行情景
- 各情景下的IRR和MOIC
- 驱动回报的关键假设
- 敏感性分析

**VIII. 风险因素**（1页）
- 按严重程度和可能性排序的关键风险
- 每个风险的缓解措施
- 交易终止风险（如有）

**IX. 推荐意见**
- 明确的推荐：继续/拒绝/有条件继续
- 关键条件或后续步骤

### 步骤3：输出格式

- 默认：带有专业格式的Word文档（.docx）
- 替代方案：用于快速审查的Markdown
- 包括财务和回报的表格，而不仅仅是文字

## 重要注意事项

- IC备忘录应客观平衡 — 诚实呈现乐观和悲观情况
- 不要最小化风险。IC成员无论如何都会发现它们；可信度很重要
- 如果用户提供，使用公司的标准备忘录模板
- 财务表格应相互关联 — 检查EBITDA桥接、资金来源与用途平衡以及回报计算是否一致
- 询问缺失的输入，而不是对交易条款或回报做出假设
