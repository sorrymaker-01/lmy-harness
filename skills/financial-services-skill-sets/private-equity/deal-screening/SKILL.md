# 交易筛选
## 工作流程

### 步骤1：提取交易事实

从提供的CIM、teaser或描述中提取：

- **公司**：名称、位置、行业/子行业
- **描述**：他们做什么（1-2句话）
- **财务数据**：收入、EBITDA、利润率、增长率
- **交易类型**：平台、附加收购、资本重组、少数股权、企业剥离
- **要价/估值**：倍数、如有说明的企业价值
- **卖方动机**：为什么现在出售
- **管理层**：留任或退出
- **关键客户**：集中度风险
- **关键风险**：明显的红旗信号

### 步骤2：对照标准筛选

应用基金的投资标准（如果未知，询问用户）：

| 标准 | 目标值 | 实际值 | 通过/失败 |
|-----------|--------|--------|-----------|
| 收入范围 | | | |
| EBITDA范围 | | | |
| EBITDA利润率 | | | |
| 增长概况 | | | |
| 行业适配性 | | | |
| 地理位置 | | | |
| 交易规模/EV | | | |
| 估值（x EBITDA） | | | |
| 客户集中度 | | | |
| 管理层连续性 | | | |

### 步骤3：快速评估

提供三部分评估：

1. **结论**：通过 / 进一步尽职调查 / 直接拒绝
2. **乐观情况**（2-3个要点）：为什么这可能是一笔好交易
3. **悲观情况**（2-3个要点）：关键风险和担忧
4. **关键问题**：首次通话需要回答的问题

### 步骤4：输出

一页纸的筛选备忘录，适合与合作伙伴分享或投资委员会快速筛选。

## 重要注意事项

- 速度很重要 — 筛选应该花费几分钟，而不是几小时
- 直接指出红旗信号。不要隐藏担忧
- 如果财务数据看起来不一致或不完整，明确标记
- 如果是首次筛选，预先询问基金的标准
- 一旦确认，将筛选标准保存在记忆中以供未来交易使用
