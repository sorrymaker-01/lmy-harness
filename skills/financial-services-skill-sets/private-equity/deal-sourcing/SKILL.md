# 交易挖掘
## 工作流程

本技能遵循三步挖掘流程：

### 步骤1：发现公司

根据用户的标准研究并识别潜在的目标公司：

- **行业/领域聚焦**：询问用户他们关注的领域（例如，"医疗保健领域的B2B SaaS"、"东南部的工业服务"）
- **交易参数**：收入范围、EBITDA范围、增长概况、地理位置、所有权类型（创始人所有、私募股权支持、企业剥离）
- **信息来源**：使用网络搜索找到符合条件的公司。查看行业报告、会议参会者名单、行业出版物和竞争格局
- **输出**：公司短名单，包含：公司名称、描述、估计收入/规模、位置、创始人/CEO姓名、网站，以及为什么它们符合投资 thesis

### 步骤2：CRM检查

在联系之前，检查公司或创始人是否已经存在于公司的CRM系统中：

- 搜索用户的电子邮件（Gmail）中与公司或创始人的先前通信
- 搜索Slack中关于目标公司的任何内部提及或先前讨论
- 询问用户："您或您的团队是否与[公司]有过任何先前的联系？"
- 标记任何现有关系、先前的拒绝或已知背景
- **输出**：对于每家公司，注明："新"（无先前联系）、"现有"（找到先前通信 — 总结），或"先前拒绝"（如果有先前拒绝的证据）

### 步骤3：起草创始人联系邮件

起草个性化的冷邮件给创始人/CEO：

- **语气**：专业但温暖。不要过于正式 — 创始人对真诚、简洁的联系回应更好
- **结构**：
  1. 简短介绍 — 你是谁和你的公司（如果不知道，询问用户公司介绍）
  2. 为什么这家公司引起了你的注意 — 引用具体内容（产品、市场地位、增长）
  3. 你在寻找什么 — 伙伴关系，而不仅仅是交易
  4. 温和的请求 — "您是否愿意进行简短的对话？"
- **个性化**：引用公司的具体产品、最新消息或市场地位。永远不要使用通用模板
- **长度**：最多4-6句话。创始人很忙
- **语气匹配**：如果用户发送过先前的联系邮件，研究它们以匹配他们的语气和风格。在Gmail中搜索带有"联系"、"介绍"、"伙伴关系"等关键词的"已发送"邮件以找到示例

### 邮件起草指南

- 主题行：保持简短具体。引用公司或行业，而不是"投资机会"
- 首次联系时不要添加附件
- 包含明确但低压力的行动召唤
- 如果可以，在Gmail中起草，否则以文本形式输出供用户复制

## 示例交互

**用户**："为我寻找德克萨斯州收入在10-50百万美元的创始人所有的工业服务公司"

**助手**：
1. 搜索网络寻找符合条件的德克萨斯州工业服务公司
2. 提供包含5-8家公司的短名单及关键详情
3. 对于每家公司，检查Gmail/Slack中的先前联系
4. 为标记为"新"的公司起草个性化联系邮件
5. 在发送前呈现草稿供用户审核

## 重要注意事项

- 在起草邮件之前，始终呈现短名单供用户审核
- 未经用户明确批准，永远不要发送邮件
- 如果用户的公司介绍或投资标准不明确，在起草前询问
- 优先考虑质量而非数量 — 5个经过充分研究的目标优于20个通用目标
