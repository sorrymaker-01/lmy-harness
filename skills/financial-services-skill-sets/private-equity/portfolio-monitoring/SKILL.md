# 投资组合监控

## 工作流程

### 步骤1：摄取财务包

- 接受用户的投资组合公司财务包（Excel工作簿、PDF或CSV）
- 提取关键财务数据：收入、EBITDA、现金余额、未偿债务、资本支出、营运资金
- 识别报告期间并与前期和预算/计划进行比较

### 步骤2：KPI提取与差异分析

要跟踪的关键指标（根据公司所在行业调整）：

**财务KPI：**
- 收入与预算对比（金额和百分比）
- EBITDA和EBITDA利润率与预算对比
- 现金余额和净债务
- 杠杆比率（净债务 / LTM EBITDA）
- 利息覆盖率
- 资本支出与预算对比
- 自由现金流

**运营KPI**（询问用户或从数据中推断）：
- 客户数量 / 每客户收入
- 员工人数 / 每员工收入
- 未完成订单 / 销售管道
- 流失率 / 留存率

### 步骤3：标记与总结

- **绿色**：在计划的5%范围内
- **黄色**：低于计划5-15% — 标记为需要讨论
- **红色**：低于计划>15%或存在契约违约风险 — 需要立即关注

输出简洁摘要：
1. 一段执行摘要（"公司X正在[超前/落后/按计划]跟踪..."）
2. 实际vs预算vs前期的KPI表格
3. 带有背景的红色/黄色标记
4. 契约合规状态（如适用）
5. 向管理层提出的问题

### 步骤4：趋势分析

如果提供多个期间：
- 随时间绘制关键指标（收入、EBITDA、现金）
- 识别趋势 — 加速、减速或稳定
- 与承保案例比较

## 重要注意事项

- 如果未提供预算/计划，始终要求提供以进行比较
- 不要假设行业特定KPI — 询问对该公司重要的指标
- 如果不知道契约水平，询问用户信贷协议条款
- 输出应适合董事会 — 简洁、事实性、无冗余
