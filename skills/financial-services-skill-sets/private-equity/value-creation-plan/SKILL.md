# 价值创造计划

## 工作流程

### 步骤1：基线评估

了解起点：
- 当前收入、EBITDA和利润率
- 组织结构和能力
- 按职能划分的关键运营指标
- 管理团队的优势和差距
- 尽职调查期间已识别的快速获胜机会

### 步骤2：价值创造杠杆

将所有杠杆映射到持有期内的EBITDA桥接：

#### 收入增长杠杆
- **有机增长**：价格上涨、销量增长、市场扩张
- **交叉销售/向上销售**：向现有客户提供新产品
- **新市场进入**：地理扩张、新垂直领域、新渠道
- **销售团队有效性**：招聘销售代表、提高转化率、缩短周期
- **并购/附加收购**：补强收购以增加收入和能力

对每个杠杆：
- 当前状态 → 目标状态
- 收入影响（$）
- 产生影响的时间线
- 所需投资
- 信心水平（高/中/低）

#### 利润率扩张杠杆
- **定价优化**：价格上涨、产品组合调整、捆绑销售
- **COGS减少**：采购节省、供应商整合、自动化
- **运营支出优化**：间接成本减少、共享服务、离岸外包
- **技术投资**：自动化、系统集成、数据分析
- **规模杠杆**：随着收入增长的固定成本杠杆

#### 战略/倍数扩张
- **平台建设**：附加收购、整合
- **经常性收入转型**：从项目转向经常性/订阅模式
- **市场定位**：品类领导地位、品牌建设
- **管理层升级**：关键招聘以专业化业务
- **ESG/治理**：董事会组建、报告改进

### 步骤4：100天计划

优先考虑交易完成后的前100天：

**第1-30天：稳定与评估**
- 管理层对齐和留任（签署雇佣协议，设定薪酬）
- 快速获胜 — 定价、明显的成本削减、低垂的果实
- 按职能划分的详细运营评估
- 客户沟通计划
- 建立报告和KPI仪表板

**第31-60天：计划与启动**
- 最终确定战略计划并向组织传达
- 启动前3-5个价值创造计划
- 开始附加并购管道开发
- 招聘关键岗位缺口
- 实施新的报告节奏（每周快报，月度审查，季度董事会）

**第61-100天：执行与衡量**
- 快速获胜计划的初步结果
- 首次董事会会议，提供运营指标
- 每个价值创造杠杆的进展报告
- 根据早期经验调整计划

### 步骤6：输出

- Word文档或PowerPoint，包含：
  - 执行摘要（1页）
  - EBITDA桥接图表
  - 价值创造杠杆详情（每个杠杆1页）
  - 100天计划时间线
  - KPI仪表板
  - 问责矩阵（谁负责什么）
- 支持EBITDA桥接的Excel模型

## 重要注意事项

- 对时间要现实 — 大多数私募股权价值创造需要12-24个月才能在财务报表中体现
- 快速获胜对 momentum 和可信度很重要，但不要过度关注成本削减而牺牲增长
- 管理层 buy-in 至关重要 — 共同制定计划，不要强加
- 跟踪计划级别的损益影响，而不仅仅是顶级EBITDA — 你需要知道什么在起作用
- 附加并购通常是最大的价值创造杠杆 — 从第1天开始构建管道
- 始终与运营合作伙伴或行业专家一起压力测试假设
