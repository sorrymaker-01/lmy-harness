# 并购模型构建器

## 工作流程

### 步骤1：收集输入

询问：
- 收购方信息（股票代码、财务数据、股价）
- 目标公司信息（财务数据、估值）
- 交易结构（现金、股票、混合）
- 协同效应假设（成本、收入、时间线）
- 融资条款（如使用债务）
- 交易费用

### 步骤2：交易结构分析

**对价分析：**
- 现金对价 = 每股价格 × 流通股数
- 股票对价 = 换股比例 × 流通股数
- 总对价 = 现金 + 股票价值 + 承担债务

**资金来源：**
- 现有现金
- 新债务（类型、利率、期限）
- 股权发行
- 卖方融资

**资金用途：**
- 股权购买价格
- 偿还目标债务
- 交易费用
- 融资费用

### 步骤3：增厚/稀释分析

**基本公式：**
- 收购方独立每股收益 = 收购方净利润 / 收购方流通股数
- 备考每股收益 = 备考净利润 / 备考流通股数
- 增厚/(稀释) = 备考每股收益 - 收购方独立每股收益
- 增厚/(稀释)百分比 = 增厚/(稀释) / 收购方独立每股收益

**备考净利润调整：**
- 目标净利润（剔除交易影响后）
- (+) 协同效应（税后）
- (-) 利息费用（新增债务）
- (+) 利息收入（使用现金）
- (-) 折旧/摊销（收购价格分配）
- (-) 交易费用（税后）
- (-) 融资费用摊销

**流通股调整：**
- 收购方现有流通股
- (+) 为交易发行的新股
- (=) 备考流通股

### 步骤4：协同效应分析

**成本协同效应类别：**
- 人员冗余（公司职能、行政）
- 设施整合（办公室、仓库）
- 采购协同（供应商谈判）
- 系统整合（IT平台）
- 销售队伍优化

**收入协同效应类别：**
- 交叉销售机会
- 地理扩张
- 新产品/服务
- 定价权

**协同效应时间线：**
- 第1年：通常实现10-30%
- 第2年：通常实现50-70%
- 第3年：完全实现

**协同效应成本：**
- 实现成本（遣散费、整合费用）
- 通常为年度协同效应的1-2倍

### 步骤5：收购价格分配

**资产公允价值调整：**
- 存货
- 固定资产
- 无形资产（客户关系、技术、商标）
- 商誉 = 购买价格 - 可辨认净资产公允价值

**摊销影响：**
- 可摊销无形资产 → 税盾
- 商誉 → 不摊销，年度减值测试

### 步骤6：敏感性分析

**关键变量：**
- 收购价格
- 协同效应金额
- 协同效应实现时间
- 股票对价比例
- 债务利率

**输出表格：**
- 不同价格和协同效应情景下的增厚/稀释
- 无协同效应时的盈亏平衡协同效应金额
- 盈亏平衡时间

### 步骤7：输出格式

**Excel模型结构：**
1. 交易摘要
2. 假设输入
3. 资金来源与用途
4. 收购价格分配
5. 备考利润表
6. 增厚/稀释分析
7. 协同效应分析
8. 敏感性分析表

## 重要说明

- 增厚/稀释分析是关键指标 — 交易是否创造价值？
- 协同效应假设应保守 — 买家经常高估协同效应
- 考虑控制权溢价 — 收购方支付溢价，期望通过协同效应收回
- 摊销费用减少净利润但提供税盾
- 商誉减值风险 — 如果交易不成功，未来可能有大额减值费用
- 股票交易中的换股比例决定所有权稀释
- 增厚/稀释分析应展示有协同效应和无协同效应的情况
