# 交易追踪器

## 工作流程

### 步骤1：设置追踪器结构

**核心列：**
- 买家名称
- 买家类型（战略买家/财务赞助商）
- 优先级层级（1/2/3）
- 主要联系人
- 联系信息

**流程状态列：**
- Teaser发送日期
- NDA发送日期
- NDA签署日期
- CIM发送日期
- 数据室访问日期
- 问题收到日期
- 问题回复日期
- 管理层会议日期
- 指示意向书收到日期
- 意向书金额
- 尽职调查状态
- 最终报价日期
- 最终报价金额
- 排他性状态

**时间表列：**
- 下一步行动
- 下一步行动日期
- 备注/评论

### 步骤2：流程阶段定义

**阶段1：流程启动**
- Teaser已发送
- 等待NDA签署
- CIM待分发

**阶段2：初步评估**
- CIM已发送
- 数据室访问已授予
- 等待问题或意向书

**阶段3：指示性投标**
- 问题已收到/已回复
- 管理层会议已安排/已完成
- 等待指示性意向书

**阶段4：尽职调查**
- 意向书已收到
- 尽职调查进行中
- 数据室完全访问

**阶段5：最终投标**
- 最终报价已收到
- 谈判进行中
- 排他性已授予

**阶段6：成交**
- 签署最终协议
- 监管审批
- 交易完成

### 步骤3：状态更新流程

**每日更新：**
- 记录所有买家沟通
- 更新状态变更
- 添加备注和评论
- 标记需要跟进的项目

**每周摘要：**
- 按阶段统计买家数量
- 识别停滞的买家
- 突出关键下一步行动
- 更新卖方

### 步骤4：关键指标追踪

**流程指标：**
- Teaser发送总数
- NDA签署率
- CIM分发数量
- 意向书收到数量
- 平均意向书倍数
- 进入第二轮的买家数量

**时间指标：**
- Teaser到NDA签署的平均天数
- CIM到意向书的平均天数
- 流程总持续时间

### 步骤5：输出格式

**Excel追踪器：**
- 按阶段分页（或使用筛选）
- 条件格式（状态颜色编码）
- 日期跟踪
- 摘要仪表板

**每周状态报告：**
- 流程概述
- 关键进展
- 活跃买家摘要
- 问题和风险
- 下周计划

## 重要说明

- 追踪器是流程的中央信息库 — 保持更新
- 定期与卖方分享状态更新
- 识别停滞的买家并主动跟进
- 跟踪所有沟通的时间戳
- 保密处理追踪器信息
- 使用追踪器识别流程瓶颈
- 最终报价前的所有报价都是非约束性的
