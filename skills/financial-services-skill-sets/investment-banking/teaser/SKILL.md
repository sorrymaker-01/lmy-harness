# 并购交易Teaser构建器

## 工作流程

### 步骤1：收集信息

询问可用的输入：
- 管理层演示文稿或CIM（如已存在）
- 财务摘要（收入、EBITDA、增长）
- 行业和商业模式描述
- 关键投资亮点
- 交易理由/目标

### 步骤2：Teaser结构

标准Teaser格式（1页）：

**标题**
- 行业/板块描述（如"领先的SaaS软件提供商"）
- "保密投资机会"

**公司概述**（2-3句话）
- 他们做什么（匿名）
- 规模和地位（如"北美领先提供商"）
- 关键差异化因素

**投资亮点**（4-6个项目符号）
- 增长轨迹
- 市场地位
- 产品/服务优势
- 客户价值主张
- 财务概况（如合适）

**财务摘要**（表格格式）
- 收入（最近3年 + LTM）
- EBITDA / EBITDA利润率
- 增长率（收入CAGR）

**交易概述**（如合适）
- 出售原因
- 交易结构（100%出售、少数股权等）
- 时间表

**联系方式**
- 投资银行联系信息
- NDA流程说明

### 步骤3：匿名化指南

**不要包含：**
- 公司名称
- 具体地点（使用区域描述，如"总部位于长三角/珠三角/京津冀地区"）
- 客户名称
- 可识别的产品名称
- 管理层姓名
- 任何允许识别公司的信息

**可以包含：**
- 行业板块
- 收入规模范围
- 员工数量范围
- 地理覆盖（区域层面）
- 一般产品描述

### 步骤4：语调和风格

- 简洁有力 — 每个字都重要
- 突出最吸引人的属性
- 使用主动动词（"引领"、"增长"、"交付"）
- 避免营销废话 — 坚持可验证的事实
- 专业设计 — 通常使用银行品牌模板

### 步骤5：输出

- PDF格式（通常1页，最多2页）
- 银行品牌模板
- 准备通过电子邮件分发

## 重要说明

- Teaser是买家看到的第一份材料 — 必须快速吸引注意力
- 匿名性至关重要 — 买家不应能从公开信息猜出公司
- 突出数字 — 高增长、强利润率、规模是关键
- 如果财务表现不佳，强调战略价值、市场地位或增长潜力
- Teaser通常发送给50-200+潜在买家
- 与卖方合作确定正确的匿名程度
