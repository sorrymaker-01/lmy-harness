# CIM构建器

## 工作流程

### 步骤1：收集源材料

询问可用的输入：
- 管理层演示文稿
- 历史财务数据（3-5年）
- 预算/预测
- 公司网站和营销材料
- 客户数据（如需要则匿名化）
- 组织架构图
- 先前的演示文稿或董事会材料
- 盈利质量报告（如有）

### 步骤2：CIM结构

标准CIM目录：

**I. 执行摘要**（2-3页）
- 公司概述 — 他们做什么，为什么他们能赢
- 投资亮点（5-7个关键卖点）
- 财务摘要 — 收入、EBITDA、增长、利润率标题数据
- 交易概述 — 出售什么，指示性时间表

**II. 公司概述**（3-5页）
- 历史和创始故事
- 使命和价值主张
- 产品和服务描述
- 商业模式和收入流
- 关键差异化和竞争优势

**III. 行业概述**（3-5页）
- 市场规模和增长动态（TAM/SAM/SOM）
- 关键行业趋势和顺风因素
- 竞争格局
- 监管环境
- 进入壁垒

**IV. 增长机会**（2-3页）
- 有机增长杠杆（新产品、新市场、定价）
- 并购/附加收购机会
- 运营改进
- 技术投资
- 白地分析

**V. 客户与销售**（3-5页）
- 客户概述（数量、细分、地理分布）
- 顶级客户分析（LOI前如需则匿名化）
- 客户集中度和留存指标
- 销售流程和市场进入策略
- 管道和未完成订单

**VI. 运营**（2-3页）
- 组织结构
- 关键人员
- 设施和地理布局
- 技术和系统
- 供应链/供应商关系

**VII. 财务概述**（5-8页）
- 历史利润表（3-5年）
- 收入分析 — 按细分市场、地理、客户类型
- EBITDA桥接和利润率分析
- 资产负债表概述
- 现金流摘要
- 资本支出历史
- 营运资金分析
- 管理层预测/预算（如包含）

**VIII. 附录**
- 详细财务报表
- 客户名单（匿名化）
- 产品目录
- 管理层简介

### 步骤3：起草指南

- **语调**：专业、实事求是、有说服力但不夸张
- **叙述**：讲述一个故事 — 为什么这家企业有吸引力、有护城河、定位增长
- **数据驱动**：用数据支持每一个主张。"强劲增长" → "收入从2021年到2024年以15%的CAGR增长"
- **视觉**：财务趋势、市场规模、竞争定位的图表和图形
- **长度**：总共40-60页 — 足够详细以告知第一轮投标，不要太长导致买家不愿阅读
- **保密性**：包含免责声明页。除非卖方批准，否则匿名化敏感客户数据

### 步骤4：输出

- Word文档（.docx），专业格式
- 单独的Excel附录，包含详细财务数据
- 图表和展品嵌入文档中

## 重要说明

- CIM是一份销售文档 — 首先展示优势，但不要隐藏重大问题（买家会在尽职调查中发现）
- 投资亮点应解决每个买家关心的3件事：增长潜力、利润率特征和护城河
- 财务正常化/备考调整应清楚标注和解释
- 与法律顾问合作处理保密免责声明和任何监管披露
- 在分发前让管理层审查事实准确性
- CIM设定估值预期 — 确保叙述支持要价
