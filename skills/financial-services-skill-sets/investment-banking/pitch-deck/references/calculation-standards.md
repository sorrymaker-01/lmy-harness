## 关键验证公式

### CAGR预测

**公式：**
```
未来值 = 现值 × (1 + CAGR)^n
```

**变量：**
- 现值：当前/基准年市场规模
- CAGR：复合年增长率（小数形式，如16.4% = 0.164）
- n：基准年与目标年之间的年数

**验证示例：**
```
来源声称：221亿美元（2024年）按16.4% CAGR = 550亿美元（2030年）

验证：22.1 × (1.164)^6 = 22.1 × 2.488 = 55.0 ✓
```

**计算n（年数）：** 计算基准年与目标年之间的年数。示例：2024→2030 = 6年，2025→2030 = 5年。

### 估值倍数

**EV/收入：**
```
EV/收入倍数 = 企业价值 ÷ 收入
隐含EV = 收入 × 倍数
```

**EV/EBITDA：**
```
EV/EBITDA倍数 = 企业价值 ÷ EBITDA
隐含EV = EBITDA × 倍数
```

**验证示例：**
```
来源声称：4.36亿美元交易，收入4500万美元，收入倍数9.7倍

验证：436 ÷ 45 = 9.69 ≈ 9.7倍 ✓
```

### 市场份额

**公式：**
```
市场份额 = (细分市场规模 ÷ 总市场规模) × 100
```

**验证示例：**
```
来源声称：在线细分市场（180亿美元）占总市场（650亿美元）的28%

验证：18 ÷ 65 = 0.277 = 27.7% ≈ 28% ✓
```

### 增长率

**同比增长：**
```
同比增长 = (当年 - 上年) ÷ 上年 × 100
```

**从端点计算CAGR：**
```
CAGR = (终值 ÷ 起始值)^(1/n) - 1
```

## 一致预期方法论

当源数据包含多个估计时，验证一致预期计算：

### 规模一致预期（范围）

**方法：** 跨所有来源的完整最小-最大范围

**示例：**
```
来源：149亿美元、183亿美元、211亿美元、212亿美元、221亿美元
一致预期：150-220亿美元（四舍五入到最接近的10亿美元）
```

### CAGR一致预期（中心集群）

**方法：** 排除异常值（最高和最低），使用中心集群范围

**示例：**
```
来源：10.6%、16.4%、17.2%、19.0%、22.7%
排除异常值：10.6%（低）、22.7%（高）
中心集群：16.4%、17.2%、19.0%
一致预期：16-19%或16-17%（保守）
```

### 预测一致预期

**方法：** 将一致预期CAGR应用于规模范围的中点

**示例：**
```
规模范围：150-220亿美元 → 中点：185亿美元
CAGR一致预期：16-17%
按16%：18.5 × (1.16)^6 = 451亿美元
按17%：18.5 × (1.17)^6 = 475亿美元
一致预期预测：450-480亿美元
```

## 舍入指南

这些是**典型约定** — 根据数值大小和模板风格调整：

| 值类型 | 典型舍入 | 示例 |
|--------|----------|------|
| 大市场规模（100亿美元+） | 最接近10亿美元 | 18.47 → 180亿美元 |
| 较小市场规模（<100亿美元） | 最接近5亿美元或1亿美元 | 2.3 → 25亿美元 |
| 规模范围 | 匹配来源精度 | 14.9-22.1 → 150-220亿美元 |
| CAGR | 整数%或0.5% | 16.4% → 16%或16.5% |
| 市场份额 | 最接近5%或匹配来源 | 27.7% → 25%或30% |
| 收入（百万美元） | 1位小数 | 18.47 → 1850万美元 |
| 倍数 | 1位小数 | 9.688 → 9.7倍 |

**舍入原则：**
- 舍入不应显著改变数值 — 对于较小值，使用更精细的精度
- 一致性比精度更重要 — 对类似数值使用相同的舍入
- 创建范围时，低端向下舍入，高端向上舍入
- 对于汇总统计（均值、中位数），匹配输入数据的精度

## 验证清单

在使用源数据中的任何计算值之前：

### 公式验证
- [ ] 预测使用正确的CAGR公式：`PV × (1 + r)^n`
- [ ] 倍数计算为EV ÷ 指标（而非反向）
- [ ] 增长率在分母中使用正确的基准年
- [ ] 百分比份额在适用时总和约为100%

### 输入验证
- [ ] 基准年数据与来源文档匹配
- [ ] CAGR/增长率与所述来源方法论匹配
- [ ] 时间段（n）正确计算
- [ ] 货币和单位一致（十亿美元vs百万美元）

### 输出验证
- [ ] 计算结果与来源所述数值匹配
- [ ] 如不匹配，调查方法论差异
- [ ] 舍入一致应用
- [ ] 结果合理（无数量级错误）

### 一致预期验证
- [ ] 所有来源包含在范围计算中
- [ ] 异常值排除方法论已记录
- [ ] 中点计算使用正确的平均
- [ ] 范围边界代表实际最小/最大值或记录的子集

## 需要调查的警示信号

**预测不匹配：**
- 计算预测与来源相差>5%
- 可能原因：不同基准年、不同CAGR或舍入

**倍数不匹配：**
- 计算倍数与来源不同
- 可能原因：不同指标定义（LTM vs NTM、收入vs净收入）

**一致预期不匹配：**
- 您的一致预期与来源的一致预期不同
- 可能原因：来源排除了某些数据点、不同的异常值处理
