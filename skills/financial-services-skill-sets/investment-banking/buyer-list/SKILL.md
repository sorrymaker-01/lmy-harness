# 买家名单构建器

## 工作流程

### 步骤1：理解交易

询问：
- 目标公司描述（行业、规模、商业模式）
- 财务概况（收入、EBITDA、增长）
- 出售理由
- 交易规模/估值预期
- 地理重点
- 卖方偏好（战略买家 vs. 财务赞助商、少数股权 vs. 控制权）

### 步骤2：战略买家识别

**战略买家类别：**
- **直接竞争对手** — 同行业的公司
- **相邻市场参与者** — 相邻板块的公司
- **垂直整合者** — 供应商或客户
- **地理扩张者** — 寻求进入新区域的公司
- **产品扩张者** — 寻求增加产品线的公司
- **技术收购者** — 寻求特定技术/能力

**研究来源：**
- 行业报告和贸易出版物
- 竞争对手分析
- 客户和供应商名单
- 行业会议参展商名单
- 年报/半年报（竞争对手的年报风险因素）
- 新闻和并购公告

### 步骤3：财务赞助商识别

**赞助商类别：**
- **平台构建者** — 在该行业有现有投资
- **附加收购者** — 寻求附加收购的平台公司
- **板块专家** — 专注于该行业的基金
- **规模匹配者** — 基金规模与交易规模匹配

**研究来源：**
- PitchBook、Preqin、Crunchbase
- 赞助商网站（投资组合公司）
- 行业并购交易历史
- PE支持的董事会任命公告
- 赞助商募资公告（新基金 = 需要部署资本）

### 步骤4：买家分类

为每个买家创建档案：

**战略买家档案：**
- 公司名称
- 股票代码（如上市）
- 总部位置
- 行业/板块
- 收入和EBITDA
- 关键高管（CEO、CFO、企业发展负责人）
- 企业发展联系人
- 战略理由（为什么他们是好买家）
- 先前并购活动
- 联系信息

**财务赞助商档案：**
- 基金名称
- 总部位置
- 基金规模 / 可投资资本
- 投资重点（行业、交易规模、地理位置）
- 相关投资组合公司
- 关键合伙人（负责该行业的合伙人）
- 先前投资活动
- 联系信息

### 步骤5：优先级排序

将买家分类为层级：
- **第一层级**：高战略契合 + 财务能力 + 已知的并购兴趣
- **第二层级**：良好的战略契合 + 财务能力
- **第三层级**：可能的战略契合或财务兴趣

优先级因素：
- 战略契合度
- 财务能力（交易规模 vs. 买家规模）
- 先前并购活动
- 文化契合
- 监管考虑
- 时间线契合

### 步骤6：输出格式

**Excel买家名单：**
- 按类型分页（战略买家、财务赞助商）
- 每个买家一行
- 筛选和排序能力
- 联系信息列
- 战略理由列
- 优先级列

**摘要备忘录：**
- 买家名单概述
- 第一层级买家分析
- 关键战略主题
- 预期竞标者
- 建议方法

## 重要说明

- 买家名单通常有50-150+个名字
- 战略买家通常比财务赞助商支付更高价格（战略溢价）
- 财务赞助商可以更快行动，监管障碍更少
- 考虑竞争动态 — 买家是否与客户/供应商竞争？
- 某些买家可能是"搅局者" — 参与竞标以获取信息
- 与卖方合作排除不应联系的买家（竞争对手、诉讼对手）
- 保密性 — 名单本身是保密的
