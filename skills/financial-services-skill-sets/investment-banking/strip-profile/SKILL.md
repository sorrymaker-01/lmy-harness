# 公司简介构建器

## 工作流程

### 步骤1：收集信息

询问：
- 公司名称和股票代码（如上市）
- 行业/板块
- 网站或公开信息来源
- 特定信息需求（如需要）

### 步骤2：信息来源

**公开来源：**
- 公司网站
- 年报
- 半年报/季报
- 投资者演示文稿
- 新闻稿
- 行业报告
- 股价数据（如上市）

### 步骤3：公司简介结构

**标准公司简介格式（1-2页）：**

**标题区域**
- 公司名称
- 股票代码（如上市）
- 总部位置
- 网站

**公司概述**（2-3段）
- 业务描述
- 历史/成立
- 市场地位
- 关键差异化因素

**财务摘要**（表格格式）
- 收入（最近3年）
- EBITDA / EBITDA利润率
- 净利润
- 员工人数
- 市值（如上市）

**业务细分**（如适用）
- 按产品/服务的收入
- 按地理的收入
- 按客户类型的收入

**关键统计数据**
- 成立年份
- 员工人数
- 运营地点数量
- 客户数量（如可用）

**关键人员**
- CEO
- CFO
- 其他关键高管

**所有权**
- 主要股东（如上市）
- 所有权结构

**近期发展**
- 关键新闻/公告
- 战略举措
- 并购活动

### 步骤4：格式标准

**文档格式：**
- Word或PDF
- 专业、简洁的布局
- 银行品牌（如适用）

**内容指南：**
- 客观、事实性的语调
- 数据驱动
- 简洁的项目符号
- 清晰的标题

**数字格式：**
- 收入：百万或十亿（一致）
- 百分比：一位小数
- 员工：整数

### 步骤5：输出

**公司简介文档：**
- 1-2页
- PDF格式
- 专业格式

**更新频率：**
- 根据需要更新
- 重大新闻后更新

## 重要说明

- 公司简介是快速参考文档 — 保持简洁
- 使用最新可用数据
- 注明数据来源和日期
- 对于上市公司，股价和市值会变化 — 注明日期
- 公司简介通常用于买家名单、董事会材料或内部研究
- 保持客观 — 不要包含投资建议
