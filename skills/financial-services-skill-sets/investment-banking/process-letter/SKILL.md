# 流程函构建器

## 工作流程

### 步骤1：理解流程

询问：
- 目标公司描述（匿名）
- 流程类型（拍卖、谈判出售、针对性流程）
- 时间表
- 保密要求
- 信息获取限制
- 买方限制（非邀约、禁止招揽等）

### 步骤2：流程函结构

**标准流程函组成部分：**

**引言**
- 识别投资银行
- 描述投资机会（匿名）
- 说明流程目的

**保密义务**
- 信息使用限制
- 禁止与第三方讨论
- 内部分发限制（需要知道原则）

**信息获取**
- 如何获取CIM
- 虚拟数据室访问
- 联系人信息

**流程指引**
- 时间表和关键日期
- 提交指示性意向书的指引
- 问题流程
- 管理层会议安排

**限制和禁止**
- 非邀约条款
- 禁止招揽员工
- 禁止招揽客户
- 静止条款

**法律条款**
- 适用法律
- 争议解决
- 责任限制

**签字栏**
- 授权代表签字
- 公司名称
- 日期

### 步骤3：NDA结构

**标准NDA组成部分：**

**双方识别**
- 披露方（卖方）
- 接收方（潜在买家）

**保密信息定义**
- 书面信息
- 口头信息
- 电子信息
- 排除内容（公开信息、独立开发等）

**保密义务**
- 使用限制（仅用于评估交易）
- 保护标准（与保护自身机密相同）
- 分发限制（需要知道的员工）
- 复制限制

**允许披露**
- 员工和顾问
- 法律要求
- 监管机构

**归还/销毁**
- 交易终止时归还材料
- 销毁副本
- 证明销毁

**期限**
- 保密义务持续时间（通常2-3年）

**限制性契约**
- 非邀约（禁止招揽客户/员工）
- 禁止招揽员工
- 禁止交易（绕过投资银行）

**法律条款**
- 适用法律
- 管辖权
- 补救措施

**签字**
- 双方签字
- 公司名称
- 日期

### 步骤4：关键条款考虑

**非邀约条款：**
- 持续时间（通常12-24个月）
- 范围（客户、供应商、员工）
- 例外情况（公开招揽、独立联系）

**禁止交易条款：**
- 防止买家绕过投资银行
- 保护银行的费用

**静止协议：**
- 限制主动收购尝试
- 通常与NDA结合

**信息访问限制：**
- 某些买家可能只能访问有限信息
- 竞争对手可能有更严格的条款

### 步骤5：输出格式

**流程函：**
- PDF格式
- 银行信头
- 专业格式
- 通常2-4页

**NDA：**
- PDF格式
- 法律文档格式
- 可签署版本
- 通常3-8页

## 重要说明

- 流程函和NDA是法律文件 — 应由法律顾问审查
- 条款因交易而异 — 根据具体情况调整
- 某些买家可能协商条款（特别是竞争对手）
- 保密义务的执行因司法管辖区而异
- 保留所有签署NDA的记录
- 某些行业有特定的监管要求（银行、医疗保健）
- 国际交易需要考虑跨境法律问题
